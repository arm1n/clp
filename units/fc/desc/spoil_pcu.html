Inboard / Outboard Power Control Units

The Power Control Units (PCUs) transfer mechanical input commands into hydraulic inputs. The input lever operates a servo valve which directs hydraulic pressure to the actuator piston. A command input moves the actuator rod with an attached feed back linkage. Actuator travel stops when the actuator rod position corresponds to the input lever position.

The PCU has four ports:
- Extend Port
- Retraction or Hold Down Port (Flight Spoiler Mode)
- Return Port
- Lift Dump Port (Ground Spoiler Mode)

The outboard spoiler PCU is powered by the No. 2 hydraulic system, No. 1 system powers the inboard spoiler PCU.




The hold down valve prevents spoiler upfloat should the actuator depressurize during flight.

The hold down valve can be manually overridden during maintenance by turning the Manual Release Control Valve (MRCV).
