LVDT, Elevator Position

There are two LVDTs located in the horizontal stabilizer; one attached to each elevator surface.
Position indication of the elevators is displayed on the Powered Flight Control System Indicator (PFCSI) shown on the Multi Function Display (MFD),and transmits the position data to the Flight Control Electronic Control Unit (FCECU)

An ELEVATOR ASYMMETRY caution light comes on when there is a difference between the left and right elevator positions.
