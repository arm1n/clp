Controls the cabin pressure schedule according to the aircraft flight profile.

The cabin pressure controller is a microprocessor based unit located in the avionics rack (top shelf). The CPC performs the following functions:
  • Provides the calculation of the cabin pressurization schedules according to aircraft flight profile
  • Provides the calculation of the control logic according to aircraft flight profile
  • Provides the control of the cabin pressure by driving the speed of the normal/auto mode channel of the electrical aft outflow valve actuator
  • Provides excessive cabin altitude warning at 10000 feet (3048 m) in auto or manual modes. For take-off or landing altitudes above 8000 ft. (2438.4 m) the warning light will come on in the conditions that follow: The aircraft take-off altitude +1000 ft. (304.8 m) Landing altitude +1000 ft. (304.8 m).
  • Exchanges information on an ARINC 429 data bus with the cabin indication module to increase monitoring efficiency
  • Initiates a POST at every power up on the ground
  • Provides continuous monitoring of bleed air
  • Provides an interface to the CDS

Post test (Max 60 sec.) Requires:
  • Controller in AUTO mode
  • RAM data not lost
  • No shop demand
  • Aircraft on GRD
