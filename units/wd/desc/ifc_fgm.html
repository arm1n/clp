FLIGHT GUIDANCE MODULE (FGM)

Each FGM module has two electronic boards as follows:
  • CPU Board (the main power supply, processing unit, ARINC and discrete inputs and outputs interfacing)
  • Extension Board (AFCS discrete and analogue inputs and output interfaces, AP and YD hardware logic)

The extension board is only necessary for the AP and YD functions. So, the FD functions can be used without the extension board, except for the TCS function. With this construction, it is possible to have a malfunction of the FD and still have AP/YD operation.

The flight guidance modules have the FD functions that follow:
  • FD mode selections
  • FD command calculations
  • Sensor selection and monitoring

Only one FGM is necessary to do the FD function.

The FGMs are Line Replaceable Modules (LRM) that are located in their related IFC.
