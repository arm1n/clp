FORMAT PUSH-BUTTON (momentary action, 1 s hold)

PUSH 1 - the NAV display on MFD changes to a FULL mode, AC centered, oriented with respect to north. Optional navigation data may be displayed in that mode, using the DATA push-button, but not the weather radar image.

PUSH 2 - the NAV display on MFD changes back to ARC mode (default selection). This partial compass mode displays ± 45° around the current heading.

PUSH and HOLD (1 s) - the NAV display changes between FMS or VOR / ILS as the navigation source.
