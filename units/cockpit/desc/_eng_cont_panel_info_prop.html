AUTOFEATHER PUSHBUTTON (alternate action)

PUSH:

  • SELECT segment (white)
  • A/F SELECT annunciation appears on the ED, just above the NP gauges

Arms autofeather system for TAKE-OFF under the following conditions:

  • power levers beyond 60° PLA
  • torques above 50%
  • condition levers at MAX/1020
  • engines running

SELECT AUTOFEATHER PUSHBUTTON (For System Test)

PUSH:

  • SELECT segment (white)

Initiates autofeather test sequence under the following conditions:

  • CONDITION levers – FUEL OFF or START/FEATHER
  • POWER levers - DISC
  • AUTOFEATHER switch - Press. Check SELECT advisory light illuminates

Observe the following messages on the ED:

  • A/F SELECT
  • A/F TEST IN PROGRESS
  • UPTRIM and ITT red radial increases
  • NTOP changes to MTOP and torque rating and torque bugs increase 10%
  • A/F ARM appears twice
  • AF TEST PASS
  • MTOP changes to NTOP and torque rating and torque bugs decrease 10% and ITTradial decreases

NOTE: If autofeather test is aborted, A/F TEST ABORT message will appear on ED. If autofeather test fails, A/F TEST FAILED message will appear on ED.

AUTOFEATHER switch - Press.

  • Check SELECT advisory out and A/F SELECT message off on ED
