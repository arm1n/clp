Index Control Panels (ICP)

The Index Control Panels (ICP) are attached to the instrument panel with four mounting screws. The fasteners and a bonding wire directly connect to the chassis to make a ground continuity connection between control panel chassis and the aircraft structure.


To control some related EFIS PFD indications, the ICP have the con- trols that follow:
SPEED BUG INDEX SETTING (two position, rotary action)
ROTATE allows the pilot to select 1 of 2 triangular speed bugs (1 outline, 1 solid) located on the PFD ASI scale
SPEED BUG ROTARY SELECTOR (rotary action)
ROTATE adjusts the speed bug value displayed on PFD airspeed indicator they are incremented "rate-aided" from 50 kt to 400 kt with a 1 knot resolution, no leading zero clockwise rotation increments the value; counterclockwise rotation decrements the value if the baro correction is selceted below the normal operating range, altitude indication becomes invalid and red ALT FAIL is displayed on the PDF
BARO SET ROTARY SELECTOR (rotary action)
ROTATE enables selection of baro correction on the PFD altitude indicator clockwise rotation increments the value; counterclockwise rotation decrements the value
DH ROTARY SELECTOR (rotary action)
ROTATE allows selection of decision height on the PFD clockwise rotation increments the value; counterclockwise rotation decrements the value
