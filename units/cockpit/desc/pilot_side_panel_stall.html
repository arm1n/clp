STALL WARN TEST SWITCH

The STALL WARN switch is set to the TEST1 position to test the functions that follow:
  • The Radio Altimeter (RA) stick pusher inhibit is functioning
  • #1 STALL SYST FAIL caution light comes on
  • PUSHER SYST FAIL caution light comes on
  • The related stick shaker operates.
  • NOTE: A stick push does not occur!

The STALL WARN switch is set to the TEST2 position to test the functions that follow:
  • The Radio Altimeter (RA) stick pusher inhibit is functioning
  • #2 STALL SYST FAIL caution light comes on
  • PUSHER SYST FAIL caution light comes on
  • The related stick shaker operates.
  • NOTE: A stick push does not occur!
