Multi-Function/Primary Flight Displays

The Flight Data Processing System (FDPS) acquires and sends data to the Electronic Instrument System (EIS). The EIS displays primary flight data, navigation, engine and system parameters on five identical and interchangeable active matrix liquid crystal.

Display Units (DU). The five DU are designated as follows:
  • Pilot’s and Copilot’s Primary Flight Displays (PFD1, PFD2)
  • Pilot’s and Copilot’s Multi Function Displays (MFD1, MFD2)
  • Engine and System Integrated Display (ED)

The two PFDs and the two MFDs comprise the Electronic Flight Instrument System (EFIS). The EFIS DU display the following flight instruments:
  • Airspeed indicator (ASI)
  • Altimeter
  • Inertial Vertical Speed Indicator (IVSI)
  • Electronic Attitude Director Indicator (EADI)
  • Electronic Horizontal Situation Indicator (EHSI)
  • Radar Altimeter
