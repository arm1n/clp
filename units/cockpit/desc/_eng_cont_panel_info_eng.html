Fuel/Engine/Propeller Control-Panel

MTOP PUSHBUTTON

  • Allows selection of maximum take off power if CLA is at 1020 andPLA is at the RATING detent.
  • When the MTOP discrete is activated, the Maximum Take- Off Power(MTOP) is selected by FADEC whenever the condition lever is in the1020 RPM position.
  • The MTOP rating is defined as the maximum available power certified for take- off operation.

RDC NP LDG PUSHBUTTON (momentary action) - Option - 803 SO 90030

PUSH - enables a reduced propeller speed for landing - Configuration for reduced NP for landing:

  • power levers between FLIGHT IDLE and approx. 50% RATING
  • with condition lever in the MIN/850 position, push the RDC NP LDG pushbutton
      • NOTE: Reduced NP Landing mode will be cancelled if condition levers are not set toMAX/1020 within 15 seconds of selecting RDC NP LDG switch.

  • ED indicates REDUCED NP LANDING
  • advance condition lever to MAX/1020; NP will remain at 850 RPM
  • a Power Lever angle of 65 degrees or greater will cancel the RDC NP selection
  • RDC NP mode can be cancelled by pushing the RDC NP LDG button again

EVENT MARKER

  • PUSH: places a “bookmark” in the Engine Monitoring System (EMS). stores a data snapshot and a data trace in the EMS for 2 minutes leading up to the event and 1 minute following the event


MCR PUSHBUTTON (momentary action)

  • PUSH: changes the engine rating associated with the 900 CLA to maximum cruise rating(MCR) 850 RPM

MCL PUSHBUTTON (momentary action)

  • PUSH: changes the engine rating associated with the MIN/850 CLA to maximum climbrating (MCL) 900 RPM

RDC TOP TRQ RESET PUSHBUTTON (momentary action

  • PUSH: resets normal take-off power

RDC TOP TRQ DEC PUSHBUTTON (momentary action)

  • PUSH: reduces NTOP requested power in steps of 2% to a limit of 10% (cannot be activated while in MTOP or MCP rating)
