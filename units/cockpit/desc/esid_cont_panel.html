ENGINE SYSTEMS INSTRUMENT DISPLAY(ESID) CONTROL PANEL

The ESCP Control Panel interfaces with the Engine Display (ED) and the Multi-Function Displays (MFD) to command the selections and reversions that follow:

  • Multi-Function Display (MFD1 and MFD2) display configuration
  • Multi-Function Display (MFD1 and MFD2) display configuration
  • Attitude and Heading Reference System (AHRS) source reversion
  • Air Data Unit source reversion
  • OFF and on brightness control for the Engine Display (ED)

The ED brightness and the reversion selection do not operate correctly :
FIM TASK 31-61-11-810-807 (TR-FIM)

The EFIS ATT/HDG SOURCE reversion selections on the ESID control panel do not operate correctly:
FIM TASK 31-61-11-810-808 (TR-FIM)

The EFIS ADC SOURCE reversion selections on the ESID control panel do not operate correctly:
FIM TASK 31-61-11-810-809 (TR-FIM)
