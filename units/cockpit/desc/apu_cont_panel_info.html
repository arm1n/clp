APU Control Panel

PWR SWITCHLIGHT- (alternate action)

PUSH - arms APU start circuits and opens the APU fuel valve,
  • only arms if aeroplane on ground, no fire detected, and EXTG switch not selected PUSH - RUN segment (out)
  • closes APU fuel valve, APU FUEL VALVE CLOSED and the APU stops FAIL - a failure is detected and the APU automatically stops

START SWITCHLIGHT- (alternate action)

PUSH - STARTER segment (amber)
  • starts automatic APU start sequence

GEN SWITCHLIGHT- (alternate action)

PUSH - ON segment (green)

APU starter-generator is supplying DC power

PUSH - STARTER segment (amber)
  • WARN segment (amber)
  • APU starter-generator off line with APU running

BL AIR SWITCHLIGHT- (alternate action)

PUSH - OPEN segment (green)
  • APU bleed air valve open
  • APU bleed air supplies air to the ECS if engine bleed air is off PUSH - OPEN segment (out)
  • APU bleed air valve is closed
  • selects APU bleed air off

GEN OHT ADVISORY LIGHT

  • APU starter-generator overheat condition APU automatically shuts down
