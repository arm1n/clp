The TCAS II system interfaces with the following:
  • ATC Transponder
  • Radio Altimeter
  • Input-Output Processor (IOP)
  • Directional Antennas
  • Audio and Radio Control Display Unit (ARCDU)
  • Configuration Module (CM)
  • Proximity Sensing Electronics Unit (PSEU)
  • Centralized Diagnostic System (CDS)
  • Input-Output Module
The TCAS II interfaces with the other systems that follow:
  • Integrated Flight Cabinet (IFC) discrete and data bus Interfaces
  • Aircraft Above Ground Level (AGL) Altitude Source Data
  • Aircraft Air/Ground and Landing Gear Up/Down Status
  • Aural warnings as output signals to the Audio Integration System (AIS)

The Audio and Radio Control Display Units (ARCDU1, ARCDU2) are used to make TCAS II mode selections.
