Do a PEC Fault Code Clear of both Engines as follows:

PEC Fault Code Clear AMM TASK 61-20-36-070-801 (TR-AMM)
1.

Set BOTH POWER LEVER to the DISC detent

2.

Move BOTH CONDITION LEVER to FUEL OFF

3.

On the Central Maintenance Panel, set MAINT DISC to ON

 4.

Push the RIG TRIM switch for at least 15 seconds

NOTE: On the caution panel, make sure the applicable PEC light comes on momentarily

 5.

Set MAINT DISC to OFF

6.

Move both POWER LEVERS and CONDITION LEVERS to the normal position.
