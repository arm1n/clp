Do a FADEC Fault Code Clear of both Engines as follows:

FADEC Fault Code Clear AMM TASK 73-21-00-070-801 (TR-AMM)
1.

Move BOTH POWER LEVER to the DISC detent

2.

Move BOTH CONDITION LEVER to MAX/1020 detent

3.

On the Central Maintenance Panel, set MAINT DISC to ON.

 4.

Push the RIG TRIM switch for at least 5 seconds.

 5.

Set MAINT DISC to OFF

6.

Move both POWER LEVERS and CONDITION LEVERS to the normal position.
