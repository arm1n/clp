As an an example: This is how you do a test of the MCL switch on the ngine Control Panel
  • Push line select key 4L to select ENGINE CONTROL.
  • On the ENGINE CONTROL panel, push and hold the MCL switch.
      • Make sure that the MCL indication on the ARCDU changes from OFF to ON for all the channels (1A, 1B, 2A, and 2B)
  • Release the MCL switch.
      • Make sure that the MCL indication changes from ON to OFF for all the channels (1A, 1B, 2A, and 2B)
