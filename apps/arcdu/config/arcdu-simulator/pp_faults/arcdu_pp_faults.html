POWERPLANT FAULTS

The POWERPLANT FAULTS PAGE on the ARCDU as follows:

  • FADEC/PEC HEALTH STATUS (Green means there is no Fault Logged, Yellow means there is Fault Logged)

  • P-PLANT Faults (Green means there is no Fault Logged, Yellow means there is Fault Logged)

  • ENGINE Faults (Green means there is no Fault Logged, Yellow means there is Fault Logged)

  • PROPELLER Faults (Green means there is no Fault Logged, Yellow means there is Fault Logged)

  • FAULT SUMMARY (Green means there is no Fault Logged, Yellow means there is Fault Logged)
