POWERPLANT MAIN MENU

The POWERPLANT MAIN MENU on the ARCDU as follows:

  • POWERPLANT FAULTS Shows all Flap Control Unit Faults

  • TREND MONITORING The Engine Monitoring System collects data on every significant event that occurs during the operation of the engine.

  • EVENT HISTORY Show different Logged events

  • OPERATIONAL DATA Operational Data Information (ENG Hours)

  • POWER ASSURANCE Use this Page to do a manual power assurance check of the engine.

  • POWERPLANT INTERFACE Operational test of different Flight Deck/Health Switches

  • EMU STATUS EMU Status Information

  • CONFIG Aircraft Configuration Information
