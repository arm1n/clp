Torquemotor current - (Engine Fault Code 264/265)

FIM TASK 61-20-06-810-829 (TR-FIM)

This fault code is generated as a result of an apparent error in the drive current to the pitch control unit servo valve, detected during power up checks. This fault could be on either the active or standby lane.

The fault isolation logic has determined that the current output varies by more than 2 mA from the test demand.

This fault code will be annuciated as a PEC caution on the ACWS. This fault code is designated as No Despatch for TLD purposes and therefore will also illuminate the powerplant message on the engine display.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Loose/dirty connectors
  • PEC
  • Pitch control unit servo valve
  • PEC wiring harness
