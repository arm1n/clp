Servo current dmd and f/b disagree - (Engine Fault Code 260)

FIM TASK 61-20-06-810-818 (TR-FIM)

This fault code is generated as a result of a disagreement between servo valve torque motor demand current (the software generated drive current) and feedback current on the active lane of the PEC. This is an indication of one of problems that follow: – that the AUPC has been triggered – that there is a problem with the PCU servo valve (or the wiring to the servo valve) – that the PEC built−in−test has failed.

NOTE: This fault code is inhibited in the event of autofeather.

The fault isolation logic has determined that the torque motor current demand differs by more than 10mA from the monitored feedback current.

This fault code will be annunciated as a PEC caution on the ACWS. This fault code is designated as No Despatch for TLD purposes and therefore will also illuminate the powerplant message on the engine display.

During fault isolation for OSG and AF test faults, the tests must be done again after corrective action has been taken. If the tests pass, a fault clear will be necessary to clear the fault in question.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Flight deck mishandling (only in conjunction with fault code 236)
  • PCU (servo valve drive coarse fault (only in conjunction with fault code 236)
  • PCU (servo valve drive fine fault)
  • PEC wiring harness problem (only in conjunction with fault code 236)
  • PEC (without fault code 236)
