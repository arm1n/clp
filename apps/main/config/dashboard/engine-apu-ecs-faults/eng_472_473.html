Invalid FADEC MAINTENANCE bit (on aircraft without Modsum 4W113558 only) - (Engine Fault Code 472/473)

FIM TASK 61-20-10-810-813 (TR-FIM)

This fault code is generated as a result of an apparent invalid FADEC MAINTENANCE signal from the engine FADEC to the PEC. The presence of this fault code might prevent certain maintenance activities from being successfully undertaken.

The fault isolation logic has determined that FADEC MAINTENANCE is invalid, as supplied by the RS422 databus from the FADEC.

The fault code is advisory and will not be annunciated to the flight crew. This fault code is designated as unlimited despatch for TLD purposes.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • ECIU
  • Airframe wiring harness (maintenance discrete to FADEC)
  • FADEC
  • Fuel control electrical wiring harness
  • PEC
