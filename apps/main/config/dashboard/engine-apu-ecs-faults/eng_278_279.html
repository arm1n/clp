PEC feather logic fault - (Engine Fault Code 278/279)

FIM TASK 61-20-06-810-828 (TR-FIM)

This task is for the Propeller Electronic Control Unit (PEC) fault code 278 − PEC Feather logic fault (On aircraft without Modsum 4−113264) or Spare (On aircraft with Modsum 4−113264). During fault isolation if any corrective action is performed a PEC fault clear must be completed. This will make sure that the problem has been fixed and the fault code does not reappear.
