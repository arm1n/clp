OVERTEMP AT RUN ( Status ) - (APU Fault Code 132-0 to 132-4)

FIM TASK 49-20-00-810-808 (TR-FIM)

Borescope Inspection of the Auxiliary Power Unit (APU)

AMM TASK 49-00-00-290-801 (TR-AMM)

The CDS shows a 132−4 OVERTEMP AT RUN message when the auxiliary power unit (APU) full authority digital electronic controller (FADEC) senses that the EGT is more than 718°C (1324°F) for three seconds or more than 788°C (1450°F) for 0.5 seconds during run mode.

The caution and warning panel (CAWP) APU and APU CONTROL panel FAIL caution lights can come on.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Fuel control assembly
  • APU FADEC
