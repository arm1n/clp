Indicated turbine temperature (ITT) related faults - (Engine Fault Code 722, 738, 805 and/or 973)

FIM TASK 73-20-00-810-807 (TR-FIM)

The fault isolation procedure that follows is to correct the full authority digital electronic control (FADEC) fault codes. The registered fault codes are 722, 738, 805 and/or 973.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Main Oil Temperature Sensor
  • Indicated Turbine Temperature (ITT) Wiring Harness
  • Control Wiring Harness (FADEC to Main Oil Temperature Sensor)
  • Terminal Block
  • Full Authority Digital Electronic Control (FADEC)
