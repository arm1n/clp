BLEED VLV STUCK CLOSED ( Status ) - (APU Fault Code 048-0 to 048-5)

FIM TASK 49-50-00-810-802 (TR-FIM)

The CDS shows the message when the Auxiliary Power Unit (APU) Full Authority Digital Electronic Controller (FADEC) senses that the bleed valve is stuck in an closed condition during the run mode.

This is a Class 2 fault. The Caution and Warning Panel (CAWP) APU and APU CONTROL panel FAIL lights do not illuminate and the APU continues to operate.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Bleed Valve
  • Bleed Valve Filter (cleaning)
