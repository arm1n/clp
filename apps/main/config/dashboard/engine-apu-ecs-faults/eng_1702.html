Engine Trend Deviation Alert - (Engine Fault Code 1702)

FIM TASK 77-31-00-810-808 (TR-FIM)

This fault code occurs when there is an important change the performance trend.

Do an analysis of the Engine Condition Trend Monitoring (ECTM) data as described in the ECTM Analytical Guide, (P/N 3043607). You can also contact your Pratt - Whitney Canada representative for advice.
