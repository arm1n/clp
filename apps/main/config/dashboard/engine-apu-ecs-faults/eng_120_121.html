Beta out of range on active lane - (Engine Fault Code 120/121)

FIM TASK 61-20-03-810-817 (TR-FIM)

This fault code is generated as a result of an apparent error in the input or output voltage levels from the PCU beta feedback transducer (BFT), on the active PEC control lane. This fault will only be detected when the propeller is rotating.

The fault accommodation logic has determined that either the excitation voltages supplied to the BFT by the PEC are outside of the normal range, or that the BFT output voltages supplied to the PEC by the BFT are outside of the normal range.

This fault code is designated as No Despatch for time limited despatch (TLD) purposes and therefore will also illuminate the POWERPLANT advisory message on the engine display. If this fault is present on both channels or if the other channel of the PEC has fault code 122/123, 124/125, 126/127 or 130/131, the PEC caution light will be illuminated on the CWAS.

NOTE: If any combination of faults 120/121, 124/125, 126/127 or 130/131 are present on both PEC channels at the same time, the propeller will automatically feather on the ground if the PLA is moved below FLIGHT IDLE.

Code 134 provides a real time status of the BFT signal validity and will clear by itself as soon as valid blade angle is detected by the affected PEC channel.

If a line replaceable unit (LRU) has to be replaced to correct this fault, make a record of all the PEC and FADEC fault codes that have been shown before its removal. Send this complete list of fault codes, indicating aircraft side (LH and RH), channel (A and B) and code number(s) to the repair shop with the returned LRU.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Loose or dirty connectors on BFT or PEC
  • Worn BFT connector pin and sockets
  • PEC wiring harness (wiring fault)
  • PCU (BFT fault)
  • PEC (internal fault)
