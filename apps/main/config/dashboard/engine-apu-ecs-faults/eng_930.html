ADU Ambient Engine Inlet Press. Vs. Ambient Engine Inlet Press. Sensor Cross Check Fault - (Engine Fault Code 930)

FIM TASK 73-20-00-810-876 (TR-FIM)

The fault isolation procedure that follows is to correct the full authority digital electronic control (FADEC) fault codes. The registered fault code is 930.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Static Air Pressure (SAP)
  • Air Data Unit (ADU)
  • Airframe Wiring Harness
