FADEC FAULT ( Status ) - (APU Fault Code 255-0 to 255-4)

FIM TASK 49-60-00-810 (TR-FIM)

The CDS shows the 255-2 FADEC FAULT message when the auxiliary power unit (APU) full authority digital electronic controller (FADEC) senses an internal RAM read/write test failed condition.

The caution and warning panel (CAWP) APU and APU CONTROL panel FAIL caution lights can come on.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • APU FADEC
