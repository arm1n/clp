Main Oil Filter IMPENDING BYPASS INDICATION - (Engine Fault Code 915)

FIM TASK 73-20-00-810-870 (TR-FIM)

If the operator finds and clears the fault code 915 during a routine line check and defers resolution within the recommended maintenance interval (Refer to FIM TASK 73-20-00-810-801 (TR-FIM)), then the main oil filter replacement is not necessary if the fault code 915 is not present at the next maintenance check.

The fault isolation procedure that follows is to correct the full authority digital electronic control (FADEC) fault codes. The registered fault code is 915.

Refer to FIM TASK 73-20-00-810-801 (TR-FIM) for a description of the type of fault code and possible deferral opportunities.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Main Oil Filter
  • Main Oil Filter Impending Bypass Switch
