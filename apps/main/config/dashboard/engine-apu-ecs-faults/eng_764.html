Derate Reset Input (Status) - (Engine Fault Code 764)

FIM TASK 73-20-00-810-834 (TR-FIM)

The fault isolation procedure that follows is to correct the full authority digital electronic control (FADEC) fault codes. The registered fault code is 764.

This fault code is flagged when the derate reset switch is selected for more then 10 seconds.

The switch is a momentary switch and should not be ON for more than 10 seconds.

The switch is located on the fuel/ engine/ propeller control panel

The switch transmits its signal to the Engine Cockpit Interface Unit (ECIU). Then, the ECIU relays the information to the FADEC using ARINC 429.

It is possible that the fault was triggered because the switch was pressed for more than 10 seconds and there are no malfunctioning components.

NOTE: In maintenance mode, the fault is triggered if the switch is pressed for more than 15 seconds.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Engine Cockpit Interface Unit (ECIU)
  • Fuel/ Engine/ Propeller Control Panel
  • Airframe Wiring Harness
