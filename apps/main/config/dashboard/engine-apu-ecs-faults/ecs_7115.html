CAB OVR TMP1 - (ECS Fault Code 7115)

CH 1 (LH) FIM TASK 21-60-00-810-862 (TR-FIM) CH 2 (RH) FIM TASK 21-60-00-810-862 (TR-FIM)
Possible Causes:
  • This fault isolation procedure is for when the central diagnostic system (CDS) shows the CAB OVR TMP1 message, Fault Code 7115. This is a DUCT HOT Detection Monitor Fault.NOTE: The acronym CAB is interchangeable with the term CABIN.
  • The caution and warning panel (CAWP) CABIN DUCT HOT caution light also comes on.NOTE: The PACK HOT lamp stays on until the condition no longer exists and the PACK Auto/Man/Off switch has been set to OFF.
  • The CDS shows this message when the system has determined the CABIN zone supply over temperature switch is open (sensed duct temperature is above 190 ±10 °F [87.8 ±5.6 °C ) and at least one of the dual CABIN pack outlet temperature sensor’s sensing elements sensed a temperature above 170.0 °F (76.7 °C) for a minimum of 5 seconds. Fault Code 7115 can also be set when only the CABIN zone supply over temperature switch goes open if both sensing elements of the CABIN pack outlet temperature sensor have failed (Fault Codes 3D01 and 3D0B). When this fault sets, the CABIN duct hot caution lamp comes on and the CABIN turbine SOV is commanded closed which shuts down the CABIN ACM.
The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):
  • ECU
  • Pack Flow Control and Shut−off Valve Filter
