Large Engine−to−Engine Parameter Deviation - (Engine Fault Code 1502)

FIM TASK 77-31-00-810-804 (TR-FIM)

This fault code occurs when important engine parameters (NH, NL, ITT, etc.) are very different from one engine to the other. This can be caused by internal engine damage or a LRU the functions incorrectly, usually a bleed valve.
