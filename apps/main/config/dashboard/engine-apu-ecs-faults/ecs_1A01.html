L BLD TMP - (ECS Fault Code 1A01)

CH 1 (LH) FIM TASK 36-10-00-810-869 (TR-FIM) CH 2 (RH) FIM TASK 36-10-00-810-869 (TR-FIM)
Possible Causes:
  • This fault isolation procedure is for when the Central Diagnostic System (CDS) shows the L BLD TMP message, fault code 1A01 for the Resistance Temperature Device (RTD) sensor.
  • The Caution and Warning Panel (CAWP) #1 BLEED HOT caution light comes on whenever the L BLD TMP message, fault code 1A01 is active.
  • The CDS shows this message when the sensed temperature is outside of the operating range of the sensor or a wiring fault has been detected.
  • Faulty left engine bleed overtemperature switch
  • FaultyECU
The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):
  • ECU
  • Bleed Overtemperature Switch
