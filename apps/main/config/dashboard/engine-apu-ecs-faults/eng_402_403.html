Uptrim state machine fault - (Engine Fault Code 402/403)

FIM TASK 61-20-09-810-812 (TR-FIM)

This fault code is generated as a result of an apparent fault with the uptrim state machine within the PEC. This is an internal PEC fault.

The fault isolation logic has determined that the uptrim state machine has not changed state after a confirmation time of 1 second after a change in the exit conditions for the state.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • PEC
