AF test aborted: loss of remote torque high in AF I−BIT - (Engine Fault Code 72/73)


FIM TASK 61-20-02-810-836 (TR-FIM)

This fault code is generated as a result of the failure of the propeller system to successfully complete an autofeather test, resulting from the apparent loss of the opposite powerplant torque signal indicating that the opposite powerplant is producing useful power.

The fault detection logic has determined that REMOTE TORQUE HI = 0 at some stage during the test.

The fault code is advisory and will not be annunciated to the flight crew. This fault code is designated as unlimited despatch for TLD purposes.

During fault isolation for OSG and AF test faults, the tests must be done again after corrective action has been taken. If the tests pass, a fault clear will be necessary to clear the fault in question.
The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):
  • Remote powerplant PEC fault
  • Local PEC fault
  • Fuel control electrical wiring harness (LH or RH)
  • Airframe wiring (cross−wing)
