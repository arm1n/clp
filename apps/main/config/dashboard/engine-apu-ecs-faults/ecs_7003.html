DORSAL LEAK - (ECS Fault Code 7003)

CH 1 (LH) FIM TASK 36-10-00-810-859 (TR-FIM) CH 2 (RH) FIM TASK 36-10-00-810-859 (TR-FIM)
Possible Causes:
  • The CDS shows the DORSAL LEAK message when the electronic control unit (ECU) senses a leak in the aft fuselage duct. The bleed duct leakage switch senses that the bleed air temperature is more then 182°C (360°F).
  • The related caution and warning panel (CAWP) #1 BLEED HOT and #2 BLEED HOT caution lights can also come on.
The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):
  • AFT Fuselage Ducts
