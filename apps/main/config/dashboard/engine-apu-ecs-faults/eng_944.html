ECIU Low Oil Pressure Lamp - (Engine Fault Code 944)

FIM TASK 73-20-00-810-886 (TR-FIM)

The fault isolation procedure that follows is to correct the Full Authority Digital Electronic Control (FADEC) fault codes. The registered fault code is 944.


The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Airframe Wiring Harness
  • Caution and Warning Panel (CAWP)
  • Engine Cockpit Interface Unit (ECIU)
  • FADEC
  • Circuit breaker H7 on the left essential DC circuit breaker panel is open
