FD TMP or CAB TMP - (ECS Fault Code 3E03)

CH 1 (LH) FIM TASK 21-60-00-810-838 (TR-FIM) CH 2 (RH) FIM TASK 36-10-00-810-801 (TR-FIM)
Possible Causes:
  • The CDS shows this message when there is a flight compartment zone temperature sensor or cabin zone temperature sensor malfunction.
  • The caution and warning panel (CAWP) can show the related FLT COMPT PACK HOT or CABIN PACK HOT caution light on.
The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):
  • Environmental Control System Electronic Control Unit (ECU)
  • Flight Compartment Zone Temperature Sensor
  • Flight Cabin Zone Temperature Sensor
