Engine Trim Mismatch Between the EEPROM and the Characterization Plug Values - (Engine Fault Code 796 and 797)

FIM TASK 73-20-00-810-889 (TR-FIM)

The fault isolation procedure that follows is to correct the Full Authority Digital Electronic Control (FADEC) fault codes. The registered fault code are 796 and 797. Code 796 is Torque Gain. Code 797 is Torque Bias. These faults will be set when the trim values that are stored in the FADEC memory are not the same as the values transmitted by the trim resistors.

The following are a list of possible malfunctioned systems and/or Line Replaceable Units (LRU): - FADEC Replacement - Characterization Plug

NOTE: Unwanted contamination in the Characterization Plug can cause a POWERPLANT message and the related 796 and 797 fault codes to come in view. Thus, it is important to keep the Characterization Plug clean.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • FADEC
  • Characterization Plug
