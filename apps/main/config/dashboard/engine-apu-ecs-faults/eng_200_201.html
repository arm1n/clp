PLABETA stuck open - (Engine Fault Code 200/201)

FIM TASK 61-20-05-810-817 (TR-FIM)

This fault code is generated as a result of of a mismatch between the power lever position inputs to the PEC. In this case the PLA beta microswitch indication is off (i.e. the PLA beta microswitch indicates flight range) but the PLA is actually in the ground range.

The fault detection logic has determined that the PLA beta microswitch is off (open) while PLA smaller 29° for a period of more than 11 seconds.

This fault code, if present on both PEC channels will be annunciated as a PEC caution on the ACWS. This fault code is designated as No Despatch for TLD (Time Limit Delay ) purposes and therefore will also illuminate the POWERPLANT message on the engine display.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • PLA Quadrant (smaller 33° switch )
  • Relay Junction box 1 (Refer to CMM 24−50−61)
  • Relay 6100−3−K2
  • Relay 6100−3−K3
  • Timing resistors DB (daughter board) No. 3, R2 ( Relay 6100−K2)
  • Timing resistors DB (daughter board) No. 3, R3 ( Relay 6100−K3)
  • PEC
  • PCU Ground Beta Enable Solenoid
  • Local essential bus or CB
  • Aircraft/Nacelle wiring
