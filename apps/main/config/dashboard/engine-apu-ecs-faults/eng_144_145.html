SyntrophyseMasterMPU bad (remote sync missing) - (Engine Fault Code 144/145)

FIM TASK 61-20-03-810-821 (TR-FIM)

This fault code is generated as a result of an apparent loss of the MPU signal from the master propeller PEC (LH) to the slave propeller PEC (RH). If this fault occurs on the left hand PEC it is a PEC fault. This fault will result in the loss of synchrophase control.

The fault isolation logic has determined that MPU 7P IN = no pulses received and SLAVE PROP = 1 for a confirmation time of 10 seconds.

The fault code is advisory and will not be annunciated to the flight crew. This fault code is designated as unlimited despatch for TLD purposes.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • LH powerplant MPU (only with fault code 162 on the LH PEC)
  • LH powerplant PEC wiring harness (only with fault code 162 on the LH PEC)
  • LH powerplant PEC
  • LH powerplant fuel control electrical wiring harness
  • Airframe wiring (cross wing communications)
  • RH powerplant fuel control electrical wiring harness
  • RH PEC
