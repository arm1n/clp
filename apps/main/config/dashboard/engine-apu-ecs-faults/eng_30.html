No Local/Other lane-FADEC Rx Data (stationary propeller ) - (Engine Fault Code 30)

FIM TASK 61-20-01-810-816 (TR-FIM)

Post Modsum 4W113558, this fault code is generated as a result of an apparent loss of data being received from the FADEC on the RS422 databus.

The fault isolation logic has determined that there is no data on the RS422 databus.

The fault code is advisory and will not be annunciated to the flight crew. This fault code is designated as unlimited despatch for TLD purposes.
The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):
  • FADEC
  • Fuel control electrical wiring harness
  • PEC
