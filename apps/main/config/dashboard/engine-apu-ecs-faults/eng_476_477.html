Invalid cockpit derate pwr incr bit (on aircraft without MS 4-113264) - (Engine Fault Code 476/477)

FIM TASK 61-20-10-810-815 (TR-FIM)

This fault code is generated as a result of an apparent indication from the FADEC that the power derate reset discrete signal is invalid. For Post Modsum 4W113558, the fault code will only be annunciated when the propeller is rotating.

The fault isolation logic has determined that POWER DERATE RESET is invalid.

The fault code is advisory and will not be annunciated to the flight crew. This fault code is designated as unlimited despatch for TLD purposes.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Engine cockpit interface unit (ECIU)
  • Engine control panel
  • Aircraft wiring harness
  • FADEC
  • Fuel control electrical wiring harness
  • PEC
