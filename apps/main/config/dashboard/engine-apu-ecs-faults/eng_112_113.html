SIM register corrupt fault - (Engine Fault Code 112/113)

FIM TASK 61-20-03-810-809 (TR-FIM)

This fault code is generated as a result of the software detecting that specified SIM registers have changed from the initial values.

The fault isolation logic has determined that for specified SIM registers, the current values have changed since initilisation.

The fault code is advisory but will not be annunciated to the flight crew as a PEC caution. This fault code is designated as No Despatch for TLD purposes and therefore will also illuminate the powerplant message on the engine display.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • PEC
