PEC28 Volt Power Not Available - (Engine Fault Code 926)

FIM TASK 73-20-00-810-873 (TR-FIM)

The fault isolation procedure that follows is to correct the full authority digital electronic control (FADEC) fault codes. The registered fault code is 926.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Control Wiring Harness (FADEC to PEC lead Ch. A or B)
  • Propeller Electronic Control (PEC)
  • Full Authority Digital Electronic Control (FADEC)
