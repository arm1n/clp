Lane identity (Status) - (Engine Fault Code 60/61)


FIM TASK 61-20-02-810-820 (TR-FIM)

This fault code is generated as a result of a failure within the PEC software to correctly identify lanes A and B within the PEC. This is an internal PEC fault.

The fault isolation logic has determined that Lane ID A or Lane ID B have contradicting values.

The fault code is advisory but will not be annunciated to the flight crew as a PEC caution. This fault code is designated as no despatch for TLD purposes and therefore will also illuminate the powerplant message on the engine display.
The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):
  • PEC
