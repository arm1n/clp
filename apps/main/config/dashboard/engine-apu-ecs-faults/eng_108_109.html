Watchdog spurious freeze - (Engine Fault Code 108/109)

FIM TASK 61-20-03-810-807 (TR-FIM)

The fault code is generated following a reset where the watchdog freeze line was unexpectedly set to freeze. This is an internal PEC fault.

The fault isolation logic has determined that following a reset the watchdog freeze line has unexpectedly frozen.

This fault code will be annunciated as a PEC caution on the ACWS. This fault code is designated as No Despatch for TLD purposes and therefore will also illuminate the powerplant message on the engine display.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • PEC
