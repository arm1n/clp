FADEC.PLA invalid - (Engine Fault Code 202/203)

FIM TASK 61-20-05-810-818 (TR-FIM)

This fault code is generated as a result of an apparent invalid PLA signal supplied from the engine FADEC to the PEC. For Post Modsum 4W113558, this fault code will only be annunciated when the propeller is rotating.

The fault detection logic has determined that the RS422 databus input from the FADEC, PLA invalid data word, is true, and the FADEC channel not in control PLA data is unavailable.

This fault code will be annunciated as a PEC caution on the ACWS. This fault code is designated as No Despatch for TLD purposes and therefore will also illuminate the powerplant message on the engine display.

During fault isolation for OSG and AF test faults, the tests must be done again after corrective action has been taken. If the tests pass, a fault clear will be necessary to clear the fault in question.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • PLA quadrant
  • Airframe wiring (PLA to FADEC)
  • FADEC
  • Fuel control electrical wiring harness
  • PEC
