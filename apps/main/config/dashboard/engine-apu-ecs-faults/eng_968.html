Derate Status Mismatch - (Engine Fault Code 968)

FIM TASK 73-20-00-810-821 (TR-FIM)

This fault code indicates that both engines are not receiving the same derate message or that one of the two engines is derated and not the other.

The FADEC receives the message through the Engine Cockpit Interface Unit (ECIU).

The most probable cause is related to operational issues.

If both engine are not in the correct conditions before selecting the Derate button, the engine which in the correct setâˆ’up will derate and not the other one. Then fault code 968 will be flagged.

If following the selection of the derate button the CLA is retarded from the 1020 position, the derate function is disabled for this engine. If the CLA of the other engine is not retarded within two seconds, fault code 968 will be flagged.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Aircraft Wiring Harness
