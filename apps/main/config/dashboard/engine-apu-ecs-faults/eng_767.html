Engine Shutdown Switch Comparison Between Channel A and B (Status) - (Engine Fault Code 767)

FIM TASK 73-20-00-810-837 (TR-FIM)

The fault isolation procedure that follows is to correct the full authority digital electronic control (FADEC) fault codes. The registered fault code is 767.

Each condition lever has 2 integrated microswitches to supply the FADEC with its actual position. The 2 positions that are sent are "shutdown" (corresponds to "fuel cutâˆ’off" on the condition lever) and "start" (corresponds to "start/ autofeather" on the condition lever).

Each FADEC channel is looking independently at one of the integrated microswitches to detect the condition lever position.

When channel "A" from the FADEC detects a different position on the condition lever from channel "B", fault code 767 is stored. This means that the 2 microswitches of one condition lever are not sending the same information to the FADEC.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Engine Shutdown Switch
  • Airframe Wiring Harness
  • Full Authority Digital Electronic Control (FADEC)
