Environmental Control System (ECS) Select Input Comparison Between Channel A and B - (Engine Fault Code 775)

FIM TASK 73-20-00-810-843 (TR-FIM)

The fault isolation procedure that follows is to correct the Full Authority Digital Electronic Control (FADEC) fault codes. The registered fault code is 775.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Engine Cockpit Interface Unit (ECIU)
  • Air condition panel
  • Airframe wiring harness
