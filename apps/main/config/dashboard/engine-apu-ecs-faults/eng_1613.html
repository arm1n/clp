Internal EMU Fault - (Engine Fault Code 1600 thru 1609 and 1613)

FIM TASK 77-31-00-810-809 (TR-FIM)

The fault isolation procedure that follows is to correct the engine monitoring unit (EMU) fault code. The registered fault codes are 1600 thru 1609 and 1613. These fault codes occur when there is an internal fault in the EMU.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):
