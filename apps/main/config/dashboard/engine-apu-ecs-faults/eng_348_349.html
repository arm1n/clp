LOW BETA lamp drive fault - (Engine Fault Code LOW BETA lamp drive fault)

FIM TASK 61-20-07-810-817 (TR-FIM)

This fault code is generated as a result of an apparent indication for the Low BETA lamp to be illuminated without being commanded. This is an internal PEC fault.

The fault isolation logic has determined that LOW PITCH (demand) is off and (LOW PITCH VMON = ON).

The fault code is advisory but will not be annunciated to the flight crew as a PEC caution. This fault code is designated as no despatch for TLD purposes and therefore will also illuminate the powerplant message on the engine display.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • PEC
