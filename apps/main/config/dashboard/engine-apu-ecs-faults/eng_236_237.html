AUPC active - (Engine Fault Code 236/237)

FIM TASK 61-20-05-810-822 (TR-FIM)

This fault code is generated as a result of the AUPC having been activated. This may not be indicative of a fault in the propeller system, but may be as a result of incorrect flight deck procedures. This fault forces the propeller to operate on the overspeed governor, and the AFM requires the pilot to shut down the powerplant under normal circumstances.

The fault isolation logic has determined that the conditions for AUPC to be activated were true.

This fault code will be annunciated as a PEC caution on the ACWS. This fault code is designated as No Despatch for TLD purposes and therefore will also illuminate the powerplant message on the engine display.

During fault isolation for OSG and AF test faults, the tests must be done again after corrective action has been taken. If the tests pass, a fault clear will be necessary to clear the fault in question.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Flight deck mishandling
  • PCU (in conjunction with fault code 260 only)
  • PEC
