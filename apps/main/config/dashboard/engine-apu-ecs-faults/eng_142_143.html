Phase index failed closed(on aircraft without Modsum 4W113558 only) - (Engine Fault Code 142/143)

FIM TASK 61-20-03-810-820 (TR-FIM)

This fault code is generated on the #2 PEC only as a result of the phase index signal to the PEC remaining high for an extended period.

The fault isolation logic has determined that for the #2 PEC, the PHASE INDEX input has remained high for a period of 80 seconds.

The fault code is advisory and will not be annunciated to the flight crew. This fault code is designated as unlimited despatch for TLD purposes.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • ANVS
  • PEC
