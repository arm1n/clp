Qb inconsistent - (Engine Fault Code 242/243)

FIM TASK 61-20-05-810-824 (TR-FIM)

This fault code is generated as a result of an apparent difference in Qb when compared with Qa and FADEC TORQUE.

The fault isolation logic has determined that Qb differs by more than 20% rated torque when compared to Qa and FADEC TORQUE. Qa and FADEC TORQUE must be within 10% rated torque of each other. This check is completed only when propeller speed is greater than 400 rpm, NP is valid and FADEC TORQUE is valid. There is a first order lag of 5 seconds before any comparison test is completed.

During fault isolation for OSG and AF test faults, the tests must be done again after corrective action has been taken. If the tests pass, a fault clear will be necessary to clear the fault in question.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Loose/Dirty connectors
  • Torque probe B
  • PEC
  • Fuel control electrical wiring harness
