High Indicated Turbine Temperature During Start - (Engine Fault Code 950)

FIM TASK 73-20-00-810-817 (TR-FIM)

FIM TASK 71-00-00-810-805 (TR-FIM)

Engine #1 (#2), Warm or Hot Start

The fault isolation procedure that follows is for an engine in a warm or hot start condition.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Battery or Aircraft Electrical Power Supply
  • Bleed Switch Selection
  • Starter System
  • Starter Generator
  • Tail Winds
  • Motive Flow Line
  • Indicated Turbine Temperature System
  • Indicated Turbine Temperature Immersion Thermocouples
  • Enviromental Control System
  • Fuel Drain Lines
  • Fuel Nozzles
  • Fuel Metering Unit
  • Full Authority Digital Electronic Control
