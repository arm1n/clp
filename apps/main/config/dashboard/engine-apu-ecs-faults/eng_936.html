Fuel Metering Unit Torque Motor - (Engine Fault Code 811 and 936)

FIM TASK 73-20-00-810-854 (TR-FIM)

The fault isolation procedure that follows is to correct the full authority digital electronic control (FADEC) fault codes. The registered fault codes are 811 and 936.

Those fault codes indicate a disrupted signal from the torque motor in the FMU.

NOTE: There are three LRUs that have a common power supply to their torque motors. A problem in the torque motor circuit of either LRU can generate fault codes for the other LRUs. When this occurs, a fault in the common circuit is detected by the FADEC and fault code 967 will be registered also. If this is the case, perform the FIM task for fault code 967 before the FIM task for fault code 811 and/or 936.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Fuel Metering Unit (FMU)
  • Control Wiring Harness
  • Full Authority Digital Electronic Control (FADEC)
