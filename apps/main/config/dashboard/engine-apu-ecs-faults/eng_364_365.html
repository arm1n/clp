Remote Lane Essential bus fault (Rotating Propeller) - (Engine Fault Code 364/365)

FIM TASK 61-20-08-810-811 (TR-FIM)

This fault code is generated as a result of the apparent loss of the remote ESSENTIAL BUS power supply to the PEC. For Post Modsum 4W113558, this fault will only be annunciated when the propeller is rotating.

The fault isolation logic has determined that the Essential Bus Monitor = 1 (There is no 28V supply from the opposite essential bus to the local PEC) for a period of 8 seconds.

The fault code is advisory but will not be annunciated to the flight crew as a PEC caution. This fault code is designated as no despatch for TLD purposes and therefore will also illuminate the poweplant message on the engine display.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Essential bus circuit breaker
  • PEC
  • Fuel control electrical wiring harness
  • Aircraft wiring
