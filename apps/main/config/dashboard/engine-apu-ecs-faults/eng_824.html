Reduce Propeller Speed (Np) Selection - (Engine Fault Code 814 and/or 824)

FIM TASK 73-20-00-810-856 (TR-FIM)

The fault isolation procedure that follows is to correct the Full Authority Digital Electronic Control (FADEC) fault codes. The registered fault codes are 814 and/or 824.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Engine Cockpit Interface Unit (ECIU)
  • Engine Control Panel
  • Airframe Wiring Harness
  • Local and remote values for RDC NP are different (814)
  • The RDC NP switch is ON over the 10 seconds limit (824)
