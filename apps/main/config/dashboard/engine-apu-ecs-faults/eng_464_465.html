Invalid FADEC Pamb - (Engine Fault Code 464/465)

FIM TASK 61-20-10-810-833 (TR-FIM)

This fault code is generated as a result of an apparent invalid ambient pressure signal from the engine FADEC to the PEC. For Post Modsum 4W113558, the fault code will only be annunciated when the propeller is rotating.

The fault isolation logic has determined that FADEC Pamb is invalid, as supplied by the RS422 databus from the FADEC.

This fault code is advisory and will not be annunciated to the flight crew. This fault code is designated as unlimited dispatch for TLD purposes.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Static air pressure (SAP)
  • Air data unit
  • Airframe wiring harness
  • FADEC
  • Fuel control electrical wiring harness
  • PEC
