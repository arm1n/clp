QSM transfer completion error - (Engine Fault Code 304/305)

FIM TASK 61-20-07-810-804 (TR-FIM)

This fault code is generated as a result of the failure of the PEC to convert analogue to digital (or digital to analogue) data within a predetermined timescale. This is an internal PEC fault.

The fault isolation logic has determined that the PEC has not completed a data conversion in 250 microseconds.

This fault code will be annunciated as a PEC caution on the ACWS. This fault code is designated as No Despatch for TLD purposes and therefore will also illuminate the powerplant message on the engine display.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • PEC
