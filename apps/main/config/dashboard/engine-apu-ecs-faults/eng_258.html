FADEC torque inconsistent - (Engine Fault Code 258)

FIM TASK 61-20-06-810-817 (TR-FIM)

This fault code is generated as a result of an apparent difference in FADEC TORQUE when compared with Qb and Qa.

The fault isolation logic has determined that FADEC TORQUE differs by more then 20% rated torque when compared to Qb and Qa. Qb and Qa must be within 10% rated torque of each other. This check is completed only when propeller speed is greater than 400 rpm, NP is valid and FADEC torque is valid. There is a first order lag of 5 seconds before any comparison test is completed.

The fault code is advisory but will not be annunciated to the flight crew as a PEC caution. This fault code is designated as no despatch for TLD purposes and therefore will also illuminate the powerplant message on the engine display.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Torque sensor
  • Control wiring harness (FADEC to torque sensor lead)
  • FADEC
  • Fuel control electrical wiring harness
  • PEC
