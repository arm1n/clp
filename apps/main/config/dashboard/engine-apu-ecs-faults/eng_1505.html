P2.2 Bleed−Off Valve (BOV) Current Range Fault (Status) - (Engine Fault Code 1505)

FIM TASK 75-31-11-810-801 (TR-FIM)

Fault code 1505 is flagged when the bleed valve torque motor, which modulates the control air supply (P3), has to operate out of its normal range. This can be caused by a blockage of the control air supply filter or a mechanical problem in the valve.

When the condition becomes more than the torque motor can correct, fault code 912 will be flagged. Fault code 912 will signal a "No−Dispatch" condition. Investigate fault code 1505 as soon as possible to avoid any disruption in operation.

Fault codes 1505 and 912 can only be detected by the FADEC when the engine is running
