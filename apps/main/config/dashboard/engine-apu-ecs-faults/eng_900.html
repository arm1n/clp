Permanent Magnet Alternator (PMA) Voltage Signal - (Engine Fault Code 900)

FIM TASK 73-20-00-810-861 (TR-FIM)

The fault isolation procedure that follows is to correct the full authority digital electronic control (FADEC) fault codes. The registered fault code is 900.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Control Wiring Harness (FADEC to PMA lead Ch. A or B)
  • Permenant Magnet Alternator (PMA)
  • Full Authority Digital Electronic Control (FADEC)
