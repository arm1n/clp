P2.2 Bleed Valve LVDT Fault (Status) - (Engine Fault Code 725)

FIM TASK 73-20-00-810-892 (TR-FIM)

The fault isolation procedure that follows is to correct the full authority digital electronic control (FADEC) fault codes. The registered fault codes is 725.

Fault code 725 indicates a metering valve LVDT system intermittent fault.

NOTE: Due to the fact that this is an intermittent fault, identifying the defective component may rely on replacing suspect LRUs and monitoring the results. If this fault code is accompanied with fault code 741 or 778, please refer to either FIM task for more detailed troubleshooting.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • P2.2 Bleed Valve
  • Full Authority Digital Electronic Control (FADEC)
  • Control Wiring Harness
