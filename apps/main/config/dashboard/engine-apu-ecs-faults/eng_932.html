RGB Scavenge Oil Filter impending bypass indication - (Engine Fault Code 932)

FIM TASK 73-20-00-810-878 (TR-FIM)

If the operator finds and clears the fault code 932 during a routine line check and defers resolution within the recommended maintenance interval (Refer to FIM TASK 73-20-00-810-801 (TR-FIM)) then filter replacement is not necessary if the fault code 932 is not present at the next maintenance check.

The fault isolation procedure that follows is to correct the full authority digital electronic control (FADEC) fault codes. The registered fault code is 932.

Refer to FIM TASK 73-20-00-810-801 (TR-FIM) for a description of the type of fault code and to AMM TASK 79-20-00-710-801 (TR-AMM) for possible deferral opportunities.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Scavenge Oil Filter
