Engine Compressor Stall Detected - (Engine Fault Code 801 and/or 933)

FIM TASK 73-20-00-810-879 (TR-FIM)

The fault isolation procedure that follows is to correct the full authority digital electronic control (FADEC) fault codes. The registered fault codes are 801 and/or 933.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • P2.2 Bleed Valve
  • P2.7 Bleed Valve
