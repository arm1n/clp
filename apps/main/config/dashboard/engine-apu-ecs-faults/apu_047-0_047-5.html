BLEED VLV STUCK OPEN ( Status ) - (APU Fault Code 047-0 to 047-5)

FIM TASK 49-50-00-810-801 (TR-FIM)

The CDS shows the message when the Auxiliary Power Unit (APU) Full Authority Digital Electronic Controller (FADEC) senses that the bleed valve is stuck in an open condition during the watch, start, run, and shutdown modes.

This is a Class 2 fault. The Caution and Warning Panel (CAWP) APU and APU CONTROL panel FAIL caution lights do not illuminate and the APU continues to operate.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Bleed Valve Filter
  • Bleed Valve
