SPEED 1 AND 2 FAULT ( Status ) - (APU Fault Code 101-0 to 101-4)

FIM TASK 49-70-00-810-812 (TR-FIM)

The CDS shows a SPEED 1 AND 2 FAULT message when the auxiliary power unit (APU) full authority digital electronic controller (FADEC) senses an internal fault or open in the speed 1 and speed 2 sensor circuits.

The CDS can also show the messages that follow:

91−5 FADEC FAULT/92−5 FADEC FAULT/93−5 FADEC FAULT/94−5 FADEC FAULT /95−5 FADEC FAULT/96−5 FADEC FAULT/97−5 FADEC FAULT/98−5 FADEC FAULT/99−5 FADEC FAULT/100−5 FADEC FAULT

The caution and warning panel (CAWP) APU and APU CONTROL panel FAIL caution lights can come on.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Speed Sensor
