PLAhigh stuck low - (Engine Fault Code 204/205)

FIM TASK 61-20-05-810-819 (TR-FIM)

This fault code is generated as a result of an apparent mismatch between the PLA HIGH discrete signal and the PLA position in the flight range, as indicated by the PLA RVDT.

The fault detection logic has determined that PLA > 70° and PLAHIGH = 0 for a period of 11 seconds.

This fault code will inhibit the autofeather advisory when PLA goes into the flight range. There is no impact on TLD. This fault is typically annunciated during the take−off roll, which will result in a rejected take−off.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • PLA microswitch
  • PEC
  • Fuel control electrical wiring harness
