ECIU Communication (engine inputs) - (Engine Fault Code 750, 754, 755 and/or 965)

FIM TASK 73-20-00-810-827 (TR-FIM)

The fault isolation procedure that follows is to correct the full authority digital electronic control (FADEC) fault codes. The registered fault codes are 750, 754, 755 and/or 965.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Full Authority Digital Electronic Control (FADEC)
  • Airframe Wiring Harness
  • Engine Cockpit Interface Unit (ECIU)
