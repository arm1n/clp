In Flight Shut Down - (Engine Fault Code 1500)

FIM TASK 77-31-00-810-802 (TR-FIM)

The Fault Isolation procedure that follows is to correct the engine monitoring unit (EMU) fault code. The registered fault code is 1500.

This fault code occurs when the engine is shut down in flight. This can be caused by engine power loss or the result of the action taken by the pilot.

NOTE: Refer to the relevant AMM sections for the necessary maintenance action. You can also contact your Pratt - Whitney Canada representative for advice.
