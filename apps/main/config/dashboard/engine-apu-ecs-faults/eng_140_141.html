PHASETRIM out of range (on aircraft without Modsum 4W113558 only) - (Engine Fault Code 140/141)

FIM TASK 61-20-03-810-819 (TR-FIM)

This fault code is generated on the #2 PEC only as a result of the phasetrim input to the PEC being outside of a predetermined range.

The fault isolation logic has determined that PHASETRIM input to the PEC is less than 0.7V or greater than 8.5V.

The fault code is advisory and will not be annunciated to the flight crew. This fault code is designated as unlimited despatch for TLD purposes.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • ANVS
  • PEC
