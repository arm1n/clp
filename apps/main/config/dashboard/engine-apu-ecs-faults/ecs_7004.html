L BLD OVR PR - (ECS Fault Code 7004)

CH 1 (LH) FIM TASK 36-10-00-810-854 (TR-FIM) CH 2 (RH) FIM TASK 36-10-00-810-801 (TR-FIM)
Possible Causes:
  • The CDS shows this message when there is a left bleed overpressure condition. The bleed overpressure switch has sensed a bleed air pressure of more than 100 psi (690 kPa).
  • The caution and warning panel (CAWP) #1 BLEED HOT or #2 BLEED HOT caution light can also come on.
