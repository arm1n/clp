SEC HX PERF - (ECS Fault Code 3502)

CH 1 (LH) FIM TASK 21-60-00-810-846 (TR-FIM) CH 2 (RH) FIM TASK 21-60-00-810-846 (TR-FIM)
Possible Causes:
  • The CDS shows this message when the environmental control system (ECS) electronic control unit (ECU) senses a difference between the performance of the heat exchanger and the specified performance limits of the heat exchanger.
  • NOTE:The level of performance of the heat exchanger is found when the temperatures from the compressor outlet, the secondary heat−exchanger outlet, and the static air temperature are compared.
  • NOTE: This test is possible only if the packs are on and the static air temperature is more than 15 °C. The pack air flow must be between the limits specified in the software. The altitude and the level of performance of the heat exchanger must be in a steady state for a minimum of five minutes.
  • This is a Class 2 fault. There is no related alert message. NOTE:It is recommended that you do the Fault Isolation, if necessary, after not more than 250 flight hours.
The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):
  • Dual Heat Exchanger
  • Secondary Heat Exchanger Outlet Temperature Sensor
  • Environmental Control System Electronic Control Unit (ECU)
