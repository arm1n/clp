Beta not calibrated. no offset stored - (Engine Fault Code 374/375)

FIM TASK 61-20-08-810-815 (TR-FIM)

This fault code is generated during beta calibration when a valid beta calibration offset from the last calibration is not retrievable from memory within the PEC. This is an internal PEC fault.

The fault isolation logic has determined that during beta calibration, it is not possible to retrieve the last calibration offset held in the PEC NVM.

The fault code is advisory but will not be annunciated to the flight crew as a PEC caution. This fault code is designated as no despatch for TLD purposes and therefore will also illuminate the powerplant message on the engine display.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • PEC
