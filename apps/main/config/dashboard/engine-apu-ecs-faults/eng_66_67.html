AF test aborted: loss of CLA permissive - (Engine Fault Code 66/67)


FIM TASK 61-20-02-810-822 (TR-FIM)

This fault code is generated as a result of the failure of the propeller system to successfully complete an autofeather test, specifically due to :
Pre Modsum 4W113558, an apparent CLA signal of 850 rpm
or
greater being present during the test.or Post Modsum 4W113558, an apparent CLA movement from Start/Feather to fuel off during the test, or an apparent CLA position greater than Start/Feather.

The fault detection logic has determined that the CLA is:
Pre Modsum 4W113558, greater than or equal to 850 rpm.
or
Post Modsum 4W113558, position has changed from Start/Feather to fuel/Off or is greater than Start/Feather.

The fault code is advisory and will not be annunciated to the flight crew. This fault code is designated as unlimited despatch for TLD purposes. This fault code does NOT change the previous status of the autofeather test pass/fail and is a diagnostic fault code to show the AF test was aborted. This fault code only shows for 5 seconds and then self clears.

During fault isolation for OSG and AF test faults, the tests must be done again after corrective action has been taken.



The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):
  • Incorrect test procedure followed
  • Incorrect CLA installation
  • PEC
  • Fuel control electrical wiring harness
  • Aircraft wiring
