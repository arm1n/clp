FD OVR TMP 2 - (ECS Fault Code 7112)

CH 1 (LH) FIM TASK 21-60-00-810-871 (TR-FIM) CH 2 (RH) FIM TASK 21-60-00-810-871 (TR-FIM)
Possible Causes:
  • This fault isolation procedure is for when the central diagnostic system (CDS) shows the FD OVR TMP2 message, Fault Code 7112. This is a DUCT HOT Detection Monitor Fault.NOTE: The acronyms FD (flight deck) and FLT COMPT (flight compartment) are interchangeable.
  • The caution and warning panel (CAWP) FLT COMPT DUCT HOT caution light will also come on.NOTE: The DUCT HOT lamp will remain illuminated until the condition no longer exists and the PACK Auto/Man/Off switch has been set to OFF.
  • The CDS shows this message when the system has determined that both sensing elements of the dual FD pack outlet temperature sensor are between 190.0 °F (87.8 °C) and 210.0 °F (98.9 °C) for 15 seconds. The setting of this fault illuminates the FD duct hot caution lamp and shuts down the FD ACM by commanding the FD turbine SOV to close. This fault code will not set if either Fault Code 7111 FD OVR TMP1 or Fault Code 7113 FD OVR TMP3 are set.
The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):
  • ECU
  • Pack Flow Control and Shut−off Valve Filter
