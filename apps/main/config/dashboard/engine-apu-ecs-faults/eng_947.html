ECIU Oil Cooler Ejector Output - (Engine Fault Code 947)

FIM TASK 73-20-00-810-888 (TR-FIM)

The fault isolation procedure that follows is to correct the full authority digital electronic control (FADEC) fault codes. The registered fault code is 947.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Airframe Wiring Harness
  • Air Cooled Oil Cooler
  • Engine Cockpit Interface Unit (ECIU)
  • Full Authority Digital Electronic Control (FADEC)
