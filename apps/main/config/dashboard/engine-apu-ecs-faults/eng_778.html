P2.2 Bleed Valve Related Faults - (Engine Fault Code 741 and/or 778)

FIM TASK 73-20-00-810-813 (TR-FIM)

The fault isolation procedure that follows is to correct the full authority digital electronic control (FADEC) fault codes. The registered fault codes are 741 and/or 778.

Fault code 741 indicates a disrupted signal from the P2.2 interstage bleed valve (refer to as "valve" in this procedure). Fault code 778 indicates a cross check fault between channel A and B.

NOTE: There are sub−systems that use common electrical connection and a fault in either one may trigger faults in several sub−systems. Therefore, if fault code 742 (PLA RVDT) AND fault code 743 (FMU LVDT) are present at the same time as fault code 741, only one sub−system or the FADEC may be found defective. If this is the case, perform an insulation resistance check and a resistance check of all three sub−systems downstream the FADEC, as indicated in the individual FIM task. If all the values are found in the specifications, replace the FADEC.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • P2.2 Interstage Bleed Valve
  • Control Wiring Harness
  • Full Authority Digital Electronic Control (FADEC)
