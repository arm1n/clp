Spool Down Trend Alert (NL) - (Engine Fault Code 1701)

FIM TASK 77-31-00-810-807 (TR-FIM)

This fault code occurs when the trend for the spool down time for the NL rotor is less than normal.

If necessary, you can get details of the event if you download the trace recording from the EMU to the laptop computer which has the Ground Based Software
