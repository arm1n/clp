Invalid FADEC RDC NP REMOTE COMMS (Modsum 4-113264) - (Engine Fault Code 456/457)

FIM TASK 61-20-10-810-810 (TR-FIM)

This fault code is generated as a result of an apparent loss of the remote power plant PEC s ability to enter the low rpm approach mode. For Post Modsum 4W113558, the fault code will only be annunciated when the propeller is rotating.

The fault detection logic has determined that the RS422 databus input from the local FADEC is indicating a communication problem with the remote powerplant.

The fault code is advisory and will not be annunciated to the flight crew. This fault code is designated as unlimited despatch for TLD purposes.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Fuel control electrical wiring harness (local or remote powerplant)
  • Cross wing electrical harness
  • Remote PEC
  • Remote FADEC
  • Local FADEC
  • Local PEC
