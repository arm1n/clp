FIM TASK 61-20-08-810-820 (TR-FIM)

This fault code is generated as a result of an apparent indication that the low beta lamp has not illuminated upon demand during the PEC power up testing.

The fault isolation logic has determined that (LOW PITCH demand = on) and (LOW PITCH VMON = off).

The fault code is advisory but will not be annunciated to the flight crew as a PEC caution. This fault code is designated as no despatch for TLD purposes and therefore will also illuminate the powerplant message on the engine display.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Caution and Warning panel
  • Fuel control electrical wiring harness
  • Aircraft wiring
  • PEC
