Low Pressure Rotor (NL) Shaft Shear Detected - (Engine Fault Code 917)

FIM TASK 73-20-00-810-872 (TR-FIM)

The fault isolation procedure that follows is to correct the full authority digital electronic control (FADEC) fault codes. The registered fault code is 917.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • NL Sensor
