Local Q Hi output fault - (Engine Fault Code 428/429)

FIM TASK 61-20-09-810-815 (TR-FIM)

This fault code is generated as a result of an apparent failure of the PEC to arm the autofeather system when the correct input torque conditions exist. This is indicative of an internal PEC fault.

The fault isolation logic has determined that the PEC software has incorrectly calculated the torque state based upon the input torque signals.
