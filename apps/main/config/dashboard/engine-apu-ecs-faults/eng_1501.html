Low Oil Pressure - (Engine Fault Code 1501)

FIM TASK 77-31-00-810-803 (TR-FIM)

The fault isolation procedure that follows is to correct the engine monitoring unit (EMU) fault code. The registered fault code is 1501.

This fault code occurs when the Low Oil Pressure switch signal is detected (Low Oil Pressure indication) but the Oil Pressure Transducer shows Normal Oil Pressure.

Do an operational check for indications of engine fault codes 772 and 916
