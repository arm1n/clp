FLT COMPT DUCT HOT or CAB OLT TMP - (ECS Fault Code 3D01)

CH 1 (LH) FIM TASK 21-60-00-810-826 (TR-FIM) CH 2 (RH) FIM TASK 21-60-00-810-827 (TR-FIM)
Possible Causes:
  • The CDS shows this message when there is a cabin compartment zone supply temperature sensor malfunction.
  • The caution and warning panel (CAWP) can show the related FLT COMPT DUCT HOT or CABIN DUCT HOT caution light on.
The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):
  • Dual Heat Exchanger
  • Secondary Heat Exchanger Outlet Temperature Sensor
  • Zone Supply Temperature Sensors
