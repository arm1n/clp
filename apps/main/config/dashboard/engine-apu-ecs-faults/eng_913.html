P2.2 Bleed Valve Torque Motor - (Engine Fault Code 779 and/or 913)

FIM TASK 73-20-00-810-831 (TR-FIM)

The fault isolation procedure that follows is to correct the full authority digital electronic control (FADEC) fault codes. The registered fault codes are 779 and/or 913. These fault codes indicate disrupted signal from the P2.2 interstage bleed valve (written as "valve" in this procedure) torque motor.

NOTE: There are three LRUs that have a common power supply to their torque motors. A problem in the torque motor circuit of either LRU can generate fault codes for the other LRUs. When this occurs, a fault in the common circuit is detected by the FADEC and fault code 967 will be registered also. If this is the case, perform the FIM task for fault code 967 before the FIM task for fault code 779 and/or 913.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • P2.2 Interstage Bleed Valve
  • Control Wiring Harness
  • Full Authority Digital Electronic Control (FADEC)
