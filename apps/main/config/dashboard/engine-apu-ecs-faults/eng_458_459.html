Abnormal RDC NP Exit - (Engine Fault Code 458/459)

FIM TASK 61-20-10-810-824 (TR-FIM)

This fault code is generated when the system exits the reduced Np function as a result of fault code 206, 454 or 456. Fault code 206 will not be recorded.

The fault detection logic has determined that one of the following conditions has been detected: – PLA smaller 50° and PLAHIGH = 1 for a period of less than 11 seconds during reduced Np operation. – That the RS422 databus input from the FADEC, FADEC REDUCED Np REQUESTED, is invalid. – That the RS422 databus input from the local FADEC is indicating a communication problem with the remote powerplant.

The fault code is advisory and will not be annunciated to the flight crew. This fault code is designated as unlimited despatch for TLD purposes.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • PLA microswitch
  • Fuel control electrical wiring harness (local or remote)
  • Cross wing electrical harness
  • Remote PEC
  • Remote FADEC
  • Local FADEC
  • Local PEC
