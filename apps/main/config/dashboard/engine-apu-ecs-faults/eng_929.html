ADU T15 Vs Engine Sensor Cross Check Fault - (Engine Fault Code 929)

FIM TASK 73-20-00-810-875 (TR-FIM)

The fault isolation procedure that follows is to correct the full authority digital electronic control (FADEC) fault codes. The registered fault code is 929.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Static Air Temperature (SAT)
  • Air Data Unit (ADU)
  • Airframe Wiring Harness
