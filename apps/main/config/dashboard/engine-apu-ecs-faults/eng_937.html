A/C Generator Chip Detector - (Engine Fault Code 937)

FIM TASK 73-20-00-810-881 (TR-FIM)

The fault isolation procedure that follows is to correct the full authority digital electronic control (FADEC) fault codes. The registered fault code is 937.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • A/C Generator
  • A/C Generator Chip Detector
  • Instrumentation Wiring Harness (A/C chip detector lead)
  • Airframe Wiring Harness
