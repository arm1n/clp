Local NP MPU extra/missing pulses - (Engine Fault Code 160/161)

FIM TASK 61-20-04-810-813 (TR-FIM)

This fault code is generated as a result of a loss of local NP MPU (dual pulse probe) signal. The fault code can also be generated as a result of a loss of synchrophasing and results in NQa being used for speed control.

The fault accommodation logic has determined that there is an inconsistency in the speed signals between the NP MPU and the gearbox torque shaft.

This fault code is designated as No Dispatch for TLD purposes and therefore will also illuminate the powerplant message on the engine display and the PEC caution light may also come on.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Slip ring assembly target screws
  • Contact between the busbar and the MPU bracket
  • MPU installation
  • MPU adjustment
  • PEC wiring harness
  • MPU
  • PEC
