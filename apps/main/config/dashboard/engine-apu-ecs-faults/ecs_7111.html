FD OVR TEMP - (ECS Fault Code 7111)

CH 1 (LH) FIM TASK 21-60-00-810-870 (TR-FIM) CH 2 (RH) FIM TASK 21-60-00-810-870 (TR-FIM)
Possible Causes:
  • This fault isolation procedure is for when the central diagnostic system (CDS) shows the FD OVR TMP1 message, Fault Code 7111. This is a DUCT HOT Detection Monitor Fault.NOTE: The acronyms FD (flight deck) and FLT COMPT (flight compartment) are interchangeable.
  • The caution and warning panel (CAWP) FLT COMPT DUCT HOT caution light will also come on.NOTE: The DUCT HOT lamp will remain illuminated until the condition no longer exists and the PACK Auto/Man/Off switch has been set to OFF.
  • The CDS shows this message when the system has determined the FD zone supply over temperature switch is open (sensed duct temperature is greater than 190 ±10 °F (87.8 ±5.6 °C) and at least one of the dual FD pack outlet temperature sensor’s sensing elements sensed a temperature greater than 170.0 °F (76.7 °C) for a minimum of 5 seconds. Fault Code 7111 can also be set when only the FD zone supply over temperature switch goes open if both sensing elements of the FD pack outlet temperature sensor have failed (Fault Codes 3D01 and 3D0B). When this fault is set, the FD duct hot caution lamp comes on and the FD ACM shuts down by commanding the FD turbine SOV closed.
