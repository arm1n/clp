FADEC Rx Checksum error - (Engine Fault Code 24)

FIM TASK 61-20-01-810-811 (TR-FIM)

This fault code is generated as a result of an apparent error in the data being received from the FADEC on the RS422 databus, indicated by an error in the checksum bit. For Post Modsum 4W113558, this fault will only be annunciated when the propeller is rotating.

The fault isolation logic has determined that there is a checksum error on the RS422 databus.

The fault code is advisory and will not be annunciated to the flight crew. This fault code is designated as unlimited despatch for TLD purposes.
The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):
  • FADEC
  • Fuel control electrical wiring harness
  • PEC
