Ignition Disable Input Comparison Between Channel A and B - (Engine Fault Code 777)

FIM TASK 73-20-00-810-845 (TR-FIM)

The fault isolation procedure that follows is to correct the full authority digital electronic control (FADEC) fault codes. The registered fault code is 777.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Engine Cockpit Interface Unit (ECIU)
  • Ignition Panel
  • Airframe Wiring Harness
