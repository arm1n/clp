Torque trim fault - (Engine Fault Code 382/383)

FIM TASK 61-20-08-810-816 (TR-FIM)

This fault code is generated as a result of an apparent change in the torque shaft bias and gain signals, as transmitted from the FADEC to the PEC, during power up.

The fault isolation logic has determined that either the torque calibration has failed, or QBTRIM or QGTRIM has varied from the previous stored values by more than 1%.
