PLA BETA stuck closed - (Engine Fault Code 198/199)

FIM TASK 61-20-04-810-815 (TR-FIM)

This fault code is generated as a result of an apparent indication of a mismatch between the power lever position indicators. In this case the PLA beta microswitch indication is on (i.e. the PLA beta should be in the beta range) but the PLA is actually in the flight range.

The fault detection logic has determined that PLA BETA = 1 and PLA > 40Â° for a period of more than 11 seconds (Pre Modsum 4W113558) or 15 seconds (Post Modsum 4W113558).

The fault code is advisory but will not be annunciated to the flight crew as a PEC caution. The fault code is designated as no despatch for TLD purposes and therefore will also illuminate the powerplant message on the engine display.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • PLA beta microswitch
  • PEC
