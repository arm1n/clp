Other lane. No Rx data - (Engine Fault Code 14/15)

FIM TASK 61-20-01-810-806 (TR-FIM)

This fault code is generated as a result of the control lane of the PEC indicating that no data is being received from the other lane of the PEC. This is an internal PEC fault. For Post Modsum 4W113558, this fault will only be annunciated when the propeller is rotating. The fault isolation logic has determined that for a period of 10 seconds it is not receiving data from the other lane.
The fault code is advisory and will not be annunciated to the flight crew. This fault code is designated as unlimited despatch for TLD purposes.
