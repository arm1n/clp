Autofeather armed logic - (Engine Fault Code 420/421)

FIM TASK 61-20-09-810-810 (TR-FIM)

This fault code is generated as a result of an apparent uncommanded illumination of the autofeather arm advisory when the propeller is rotating.

The fault isolation logic has determined that (AF armed lamp = on) AND NOT (AF state = AFFAI or AF state = AFARM) OR (AF armed lamp disabled AND AF armed lamp = on) AND NOT NO PROP ROTATION.

This fault code will inhibit the autofeather advisory when PLA goes into the take−off range. There is no impact on TLD. This fault is typically annunciated during the take−off roll, which will result in an rejected take−off.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • On aircraft without MS 4−113264 flight crew initiated IOM/IOP reset
  • PEC
