P3 SIGNAL RELATED FAULT (STATUS) - (Engine Fault Code 724)

FIM TASK 73-20-00-810-894 (TR-FIM)

The fault isolation procedure that follows is to address the Full Authority Digital Electronic Control (FADEC) fault codes. The registered fault code is 724.

This fault code indicates a P3 signal intermittent fault.

Refer to FIM TASK 73-20-00-810-801 (TR-FIM) for a description of the type of fault codes and possible deferral opportunities.

Because this is an intermittent fault, identification of the defective component can rely to the replacement of the suspect Line Replaceable Units (LRU) and monitoring the results. If this fault code is accompanied with fault codes 740, 789 and/or 790, please refer to FIM TASK 73-20-00-810-812 (TR-FIM) for more detailed troubleshooting.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • P3 transducer
  • Control wiring harness (FADEC to P3 transducer)
  • FADEC
