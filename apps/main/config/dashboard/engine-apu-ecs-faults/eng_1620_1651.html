Internal EMU Fault - (Engine Fault Code 1620 thru 1651 and 1657)

FIM TASK 77-31-00-810-810 (TR-FIM)

The fault isolation procedure that follows is to correct the engine monitoring unit (EMU) fault code. The registered fault codes are 1620 thru 1651 and 1657.

These fault codes occur when there is an internal fault in the EMU. This could be caused by a hardware problem or memory corruption.

Do an EMU memory clear

If the fault occurs again, replace the EMU at a convenient time

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):
