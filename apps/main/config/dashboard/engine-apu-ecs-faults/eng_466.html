Invalid FADEC QbTrim - (Engine Fault Code 466)

FIM TASK 61-20-10-810-834 (TR-FIM)

This fault code is generated as a result of an apparent invalid FADEC Qb trim signal from the engine FADEC to the PEC. Qb signal is a signal representing engine torque shaft bias. For Post Modsum 4W113558, the fault code will only be annunciated when the propeller is rotating.

The fault isolation logic has determined that FADEC Qb trim is invalid, as supplied by the RS422 databus from the FADEC.

The fault code is advisory and will not be annunciated to the flight crew. This fault code is designated as unlimited despatch for TLD purposes.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • FADEC mounted characterization plug P3
  • FADEC
  • Fuel control electrical wiring harness
  • PEC
