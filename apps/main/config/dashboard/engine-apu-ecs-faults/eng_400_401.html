Autofeather state machine fault - (Engine Fault Code 400/401)

FIM TASK 61-20-09-810-806 (TR-FIM)

This fault code is generated as a result of the failure of the PEC autofeather logic to correctly change its state, in response to changes in external parameters that should result in autofeather state change.

The fault detection logic has determined that there is an internal fault in the PEC resulting in an incorrect setting of ARM or STANDBY when the logic requests the other state.

During fault isolation for OSG and AF test faults, the tests must be done again after corrective action has been taken. If the tests pass, a fault clear will be necessary to clear the fault in question.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • PEC
