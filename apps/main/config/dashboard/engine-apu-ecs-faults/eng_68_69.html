AF test aborted: loss of PLA permissive - (Engine Fault Code 68/69)


FIM TASK 61-20-02-810-833 (TR-FIM)

This fault code is generated as a result of the failure of the propeller system to successfully complete an autofeather test, directly resulting from an apparent PLA lever position at or above flight idle during the test.

Pre Modsum 4W113558, the fault isolation logic has determined that PLA HI = 1 (this signal is taken from a microswitch underneath the PLA). Post Modsum 4W113558, the logic has determined that PLA HI=1 and/or the PLA angle is outside the range of 15 degrees to 25 degrees (ground idle plus or minus 5 degrees).

The fault code is advisory and will not be annunciated to the flight crew. This fault code is designated as unlimited despatch for TLD purposes. This fault code does NOT change the previous status of the autofeather test pass/fail and is a diagnostic fault code to show the AF test was aborted. This fault code only shows for 5 seconds and then self clears.

During fault isolation for OSG and AF test faults, the tests must be done again after corrective action has been taken.
The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):
  • Incorrect test procedures followed (inappropriate PLA position during the test)
  • PLA microswitch
  • PEC
  • FADEC (Post Modsum 4W113558 only)
  • Fuel Control Electrical Wiring Harness (Post Modsum 4W113558 only)
