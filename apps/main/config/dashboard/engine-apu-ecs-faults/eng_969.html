No Light Up with Igniter No. 1 - (Engine Fault Code 969)

FIM TASK 73-20-00-810-822 (TR-FIM)

The FADEC commands ignition either from channel A or channel B in successive turns during normal operation mode.

The fault detection logic uses a counter. The FADEC gives points to each channel separately based on the ignition results. Each successful start will decrease the counter by 1 point. Each ignition command that resulted in no engine light−up will attribute 3 points to the channel. The minimum number of points is 0 and the maximun is 9. The FADEC does not count points outside those limits.

The fault code will be triggered for a specific channel when it accumulates 9 points.

After a channel has reached 9 points, it will then be used systematically by the FADEC to light the engine. It will use this channelto light the engine until it reaches 0 point.

For Pre−SB35140, if you replace the ignition cables or if there is damage, then you must replace the igniter plugs also. This is because an internal leak in the plug can damage the cables. The leak in the plug is not necessarily visible or detectable.

Detachment of the igniter floating collar from the combustion chamber can result in the inability to light the engine with active igniter plug. It is recommended to inspect the igniter ports (both) to validate the presence of the igniter floating collar prior to igniter plug installation.

In order to differentiate ignition system channels A and B from FADEC channels A and B, this procedure refers to ignition system channel A as channel No. 1 and ignition system channel B as channel No. 2.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Igniter Plugs
  • Ignition Exciter
  • Ignition cables
  • Combustion Chamber Liner Igniter Floating Collar(s)
