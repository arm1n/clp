Remote AF permission stuck high in AF I−BIT - (Engine Fault Code 290)

FIM TASK 61-20-06-810-838 (TR-FIM)

This fault code is generated as result of an apparent failure of the opposite PEC to grant an autofeather permission signal during the autofeather test.

The fault isolation logic has determined that REMOTE AF PERMISSION = 0 at some stage during the 5 second autofeather confirmation period

The fault code is advisory and will not be annunciated to the flight crew. This fault code is designated as unlimited dispatch for TLD purposes.

During fault isolation for OSG and AF test faults, the tests must be done again after corrective action has been taken. If the tests pass, a fault clear will be necessary to clear the fault in question.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Opposite powerplant PEC
  • Fuel control electrical wiring harness
  • Airframe wiring harness (cross−wing)
  • Local powerplant PEC
