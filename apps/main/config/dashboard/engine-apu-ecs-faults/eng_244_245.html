Qa and Qb inconsistent - (Engine Fault Code 244/245)

FIM TASK 61-20-05-810-825 (TR-FIM)

This fault code is generated as a result of an apparent difference of more than 20% between the two torque probe signals supplied to the PEC.

The fault isolation logic has determined that QA and QB differ by more than 20% when the propeller speed is greater than 400 rpm. There is a lag of up to 10 seconds before any comparison test is completed.

  • Torque probes A or B
  • Fuel control electrical wiring harness
