CAB OVR TMP 4 - (ECS Fault Code 7118)

CH 1 (LH) FIM TASK 21-60-00-810-865 (TR-FIM) CH 2 (RH) FIM TASK 21-60-00-810-865 (TR-FIM)
Possible Causes:
  • This fault isolation procedure is for when the central diagnostic system (CDS) shows the CAB OVR TMP4 message, Fault Code 7118. This is a DUAL DUCT HOT Detection Monitor Fault.NOTE: The acronym CAB is interchangeable with the term CABIN.
  • The caution and warning panel (CAWP) FLT COMPT and CABIN DUCT HOT caution lamps also come on.NOTE: The PACK HOT lamp stays on until the condition no longer exists and the PACK Auto/Man/Off switch has been set to OFF.
  • The CDS shows this message when the system has determined that the CABIN zone supply over temperature switch is open (sensed duct temperture is greater than 190±10 °F (87.8 ±5.6 °C ) and at least one of the dual CABIN pack outlet temperature sensors sensed a temperature greater than 190.0 °F (87.8 °C) for a minimum of 20 seconds. Fault Code 7118 can also set if either Fault Code 7115, Fault Code 7116, or Fault Code 7117 were set and stayed set for a minimum of 20 seconds after the CABIN turbine SOV was commanded closed. When this fault is set, both the FLT COMPT and CABIN DUCT HOT cautions lamps come on and the pack flow control valve is commanded closed to stop airflow to the ACMs and pack bypass valves.
The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):
  • ECU
  • Pack Bypass Valve Filter
  • Pack Flow Control and Shut−off Valve Filter
