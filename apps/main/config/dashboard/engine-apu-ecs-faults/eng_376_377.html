Beta not calibrated. permissives lost during cal - (Engine Fault Code 376/377)

FIM TASK 61-20-08-810-812 (TR-FIM)

This fault code is generated when a beta calibration has failed to complete and no other related faults (Fault codes 374 and 378) are generated. This is an internal PEC fault.

The fault isolation logic has determined that the beta calibration has not completed and that fault codes 374 and 378 have not been generated.

The fault code is advisory but will not be annunciated to the flight crew as a PEC caution. This fault code is designated as no despatch for TLD purposes and therefore will also illuminate the powerplant message on the engine display.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • PEC
