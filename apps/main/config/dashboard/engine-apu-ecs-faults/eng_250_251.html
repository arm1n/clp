Q1 valid fault - (Engine Fault Code 250/251)

FIM TASK 61-20-06-810-808 (TR-FIM)

This fault code is generated as a result of an apparent torque processing problem within the PEC. This is an internal PEC fault.

The fault isolation logic has determined that either Q1 is valid when NQA larger 120 rpm, or that Q1 is invalid when NQA > 120 rpm.
