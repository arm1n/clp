Last AUPC test failed - (Engine Fault Code 220/221)

FIM TASK 61-20-05-810-831 (TR-FIM)

This fault code is generated as a result of a failure of the propeller system to complete an AUPC test (during the autofeather/uptrim test).

Pre Modsum 4W113558, the fault isolation logic has determined that either AUPC TEST STATUS = 0 (test failed) or AUPC TEST ABORT = 1 (test aborted). Post Modsum 4W113558, the logic has determined that AUPC TEST STATUS = 0 (last test failed) and AUPC TEST ABORT = 0 (test not aborted).

The fault code is advisory and will not be annunciated to the flight crew. For Pre Modsum 4W113558, this fault code is designated as no dispatch for TLD purposes and therefore will also illuminate the powerplant message on the engine display. For Post Modsum 4W113558, this fault code is designated as unlimited dispatch for TLD purposes, provided that repairs to correct and clear the fault are completed within 10 days of fault detection or within 150 flight hours of fault occurrence, whichever occurs later.
