Invalid FADEC REMOTE RECEIVED UPTRIM A & B bit - (Engine Fault Code 482/483)

FIM TASK 61-20-10-810-818 (TR-FIM)

This fault code is generated as a result of an apparent indication from remote FADEC that the uptrim confirmation signal from the remote powerplant is invalid. For Post Modsum 4W113558, the fault code will only be annunciated when the propeller is rotating.

The fault isolation logic has determined that FADEC REMOTE RECEIVED UPTRIM A and B is invalid.

The fault code is advisory and will not be annunciated to the flight crew. This fault code is designated as unlimited dispatch for TLD purposes.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Local PEC
  • Remote FADEC
  • ECIU
  • Fuel control electrical wiring harness
  • Local FADEC
  • Aircraft wiring (FADEC to FADEC)
