Invalid FADEC CAS - (Engine Fault Code 460/461)

FIM TASK 61-20-10-810-832 (TR-FIM)

This fault code is generated as a result of an apparent out of range/invalid value of airspeed, as supplied to the PEC by the FADEC. The data has apparently been declared invalid by the FADEC. For Post Modsum 4W113558, the fault code will only be annunciated when the propeller is rotating

The fault detection logic has determined that FADEC CAS is invalid as supplied via the R422 databus.

The fault code is advisory and will not be annunciated to the flight crew. This fault code is designated as unlimited dispatch for TLD purposes.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Flight data processing system (FDPS)
  • Fuel control electrical wiring harness
  • FADEC
  • PEC
