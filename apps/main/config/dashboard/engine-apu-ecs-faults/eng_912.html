P2.2 Bleed Valve Position - (Engine Fault Code 912 and 918)

FIM TASK 73-20-00-810-830 (TR-FIM)

The fault isolation procedure that follows is to correct the full authority digital electronic control (FADEC) fault codes. The registered fault codes are 912 and 918. These faults can only be detected by the FADEC when the engine is running.

Fault code 912 indicates that the valve piston does not move to the correct position. This can be caused by a problem with the control air supply (P3) or a mechanical malfunction in the valve.

Fault code 918 is a flag that indicates that the P2.7 BOV was opened by the FADEC because there was not sufficient bleed air from the P2.2 BOV.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • P2.2 Bleed Valve Screen Cartridge
  • P2.2 Bleed Valve
  • P3 Air Supply Tubes
  • P3 Air Separator
