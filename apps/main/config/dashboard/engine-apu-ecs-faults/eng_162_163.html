NP inconsistent with NQa and NQb speeds - (Engine Fault Code 162/163)

FIM TASK 61-20-04-810-814 (TR-FIM)

This fault code is generated as a result of a difference between the propeller NP MPU speed signal and the torque shaft Qa and Qb torque probe speed signals. This fault code forces the propeller to operate at the overspeed governor setting and the AFM requires the pilot to shut down the powerplant under normal circumstances (Pre Modsum 4W113558). The fault code causes a loss of synchrophasing and results in NQa being used for speed control (Post Modsum 4W113558).

The fault accommodation logic has determined that there is a difference of more than 25 rpm between the propeller MPU NP and each of the gearbox mounted torque probe Qa and Qb speeds.

This fault code will be annunciated as a PEC caution on the ACWS (Pre Modsum 4W113558 only). This fault code is designated as No Despatch for TLD purposes and therefore will also illuminate the powerplant message on the engine display.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Slip ring assembly target screws
  • Contact between the busbar and the MPU bracket
  • MPU installation
  • MPU adjustment
  • PEC wiring harness
  • MPU
  • PEC
  • Torque shaft
  • Torque probes Qa and Qb
