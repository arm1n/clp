Slave propeller fault - (Engine Fault Code 62/63)


FIM TASK 61-20-02-810-832 (TR-FIM)

This fault code is generated as a result of the PEC that is designated as the “slave” for the purpose of synchrophasing apparently failing to indicate that it is the slave PEC (Slave is the RH powerplant). This fault should only ever occur on the RH powerplant, if it occurs on the LH powerplant, replace the PEC.

The fault isolation logic has determined that with Np > 500 rpm, SLAVE PROPELLER = 0 on the slave PEC.

The fault code is advisory and will not be annunciated to the flight crew. This fault code is designated as unlimited despatch for TLD purposes.
The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):
  • PEC
  • Fuel control electrical wiring harness
  • Firewall − wing root harness
  • Airframe wiring
