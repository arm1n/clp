Invalid FADEC in sync bit - (Engine Fault Code 488/489)

FIM TASK 61-20-10-810-819 (TR-FIM)

This fault code is generated as a result of an apparent indication from the FADEC that the RS422 data is out of synchronization. For Post Modsum 4W113558, the fault code will only be annunciated when the propeller is rotating.

The fault detection logic has determined that the FADEC in sync bit is invalid.

The fault code is advisory and will not be annunciated to the flight crew. This fault code is designated as unlimited dispatch for TLD purposes.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • FADEC
  • Fuel control electrical wiring harness
  • PEC
