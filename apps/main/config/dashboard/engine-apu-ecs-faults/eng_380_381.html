Beta not calibrated . offset too large - (Engine Fault Code 380/381)

FIM TASK 61-20-08-810-813 (TR-FIM)

This fault code is generated as a result of an apparent variation in the beta calibration angle which is outside limits. This is a check of the hydraulic flight fine stop.

The fault isolation logic has determined that during beta calibration, the feedback of blade angle from the Pitch Control Unit beta feedback device is outside a window of 7° to 19°.

The fault code is advisory but will not be annunciated to the flight crew as a PEC caution. This fault code is designated as no despatch for TLD purposes and therefore will also illuminate the powerplant message on the engine display.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Beta tubes rigging
  • PCU BFT
  • Beta tubes (magnetized)
  • PEC
