Other lane. No FADEC data - (Engine Fault Code 10/11)

FIM TASK 61-20-01-810-804 (TR-FIM)

This fault code is generated as a result of an apparent loss of FADEC data to the PEC lane that is not in control of the propeller. For Post Modsum 4W113558, this fault will only be annunciated when the propeller is rotating.
The fault isolation logic has determined that, for a period of 10 seconds, no data is being received by the other lane. The fault code is advisory and will not be annunciated to the flight crew. This fault code is designated as unlimited despatch for TLD purposes.



The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):
  • PEC
  • FADEC
  • Fuel control electrical wiring harness
