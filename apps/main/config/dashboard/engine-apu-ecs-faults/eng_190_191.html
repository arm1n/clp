CLA unavailable for control - (Engine Fault Code 190/191)

FIM TASK 61-20-04-810-820 (TR-FIM)

This fault code is generated as a result of an apparent fault on the CLA signal to the PEC control lane and the unavailability of the CLA signal from the PEC standby lane.

The fault isolation logic has determined that, for a period of 1 second (Pre Modsum 4W113558) or 5 seconds (Post MOdsum 4W113558), the local CLA input is faulty, and the standby lane CLA signal is unavailable.

The fault code is advisory but will not be annunciated to the flight crew as a PEC caution. This fault code is designated as no dispatch for TLD purposes and therefore will also illuminate the powerplant message on the engine display.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • PEC
  • CLA condition quadrant
