NVM inconsistent (possible data loss) - (Engine Fault Code 86/87)


FIM TASK 61-20-02-810-826 (TR-FIM)

This fault code is generated as a result of the failure of the PEC to retrieve values from its own Non Volatile Memory (NVM). This is an internal PEC fault.

The fault isolation logic has determined that the software cannot retrieve valid values from the NVM at initialisation.

The fault code is advisory but will not be annunciated to the flight crew as a PEC caution. This fault code is designated as No Despatch for TLD purposes and therefore will also illuminate the powerplant message on the engine display.
The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):
  • PEC
