WoW A and WoW B disagree - (Engine Fault Code 340)

FIM TASK 61-20-07-810-813 (TR-FIM)

This fault code is generated as a result of an apparent contradiction between the weight on wheels lane A signal and weight on wheels lane B signal. For Post Modsum 4W113558 this fault will only be annunciated when the propeller is rotating.

The fault isolation logic is indicating that weight on wheels A is different from weight on wheels B.

The fault code is advisory but will not be annunciated to the flight crew as a PEC caution. This fault code is designated as no dspatch for TLD purposes and therefore will also illuminate the POWERPLANT message on the engine display.

This fault code may be generated by faulty radio altimeter signals in flight. This will also illuminate the POWERPLANT message on the engine display.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • PSEU
  • Fuel control electrical wiring harness
  • Fuel control electrical wiring harness cleanliness/tightness
  • PEC
  • Radio Altimeter
