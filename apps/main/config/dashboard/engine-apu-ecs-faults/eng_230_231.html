FIM TASK 61-20-05-810-828 (TR-FIM)

This fault code is generated during the overspeed governor test or during the automated flight fine stop test (Post Modsum 4W113558), as a result of the flight fine stop apparently being out of limits.

The fault detection logic has determined that either BETALIM 1 smaller 13 degrees or BETALIM 2 > 19 degrees.

The fault code is advisory and will not be annunciated to the flight crew. This fault code is designated as unlimited dispatch for TLD purposes.

During fault isolation for OSG and AF test faults, the tests must be done again after corrective action has been taken. If the tests pass, a fault clear will be necessary to clear the fault in question.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Beta tube rigging
  • PCU
  • PEC
