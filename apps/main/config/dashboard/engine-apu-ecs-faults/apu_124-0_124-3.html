DECELERATION ( Status ) - (APU Fault Code 124-0 to 124-3)

FIM TASK 49-20-00-810-806 (TR-FIM)

The CDS shows a 124−3 DECELERATION message when the auxiliary power unit (APU) full authority digital electronic controller (FADEC) senses that the APU acceleration is less than −0.5% per second during the start mode (deceleration).

The caution and warning panel (CAWP) APU and APU CONTROL panel FAIL caution lights can come on.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • APU FADEC
