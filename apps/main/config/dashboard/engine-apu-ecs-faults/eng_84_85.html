NVM write/read failure (Status) - (Engine Fault Code 84/85)


FIM TASK 61-20-02-810-821 (TR-FIM)

This task is for the Propeller Electronic Control Unit (PEC) fault code 84 − NVM write/read failure (On aircraft without Modsum 4−113264) or NVM read/write failure (On aircraft with Modsum 4−113264). This fault code is generated as a result of unsuccessful attempts to write to the NVM during initialisation. This is an internal PEC fault.

The fault isolation logic has determined that the PEC could not write to the NVM.

The fault code is advisory but will not be annunciated to the flight crew as a PEC caution. This fault code is designated as No Despatch for TLD purposes and therefore will also illuminate the powerplant message on the engine display.
