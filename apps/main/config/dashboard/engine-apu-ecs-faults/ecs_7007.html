L ANA AFT LEAK - (ECS Fault Code 7007)

CH 1 (LH) FIM TASK 36-11-46-810-801 (TR-FIM) CH 2 (RH) FIM TASK 36-11-46-810-802 (TR-FIM)
Possible Causes:
  • The CDS shows this message when the environmental control system (ECS) electronic control unit (ECU) analog channel senses a condition that follows:
  • Leak in the aft fuselage duct. The bleed duct leakage switch senses that the bleed air temperature is more than 182°C (360°F).
  • Bleed duct leakage switch has failed.
  • The caution and warning panel (CAWP) #1 BLEED HOT or #2 BLEED HOT caution light can also come on.
The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):
  • Aft Fuselage Duct
