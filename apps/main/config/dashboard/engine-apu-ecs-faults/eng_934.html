Shutdown Failure - (Engine Fault Code 934)

FIM TASK 73-20-00-810-880 (TR-FIM)

The fault isolation procedure that follows is to correct the full authority digital electronic control (FADEC) fault codes. The registered fault code is 934.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Airframe Wiring Harness
  • Control Wiring Harness (FADEC to NH sensor lead Ch. A or B or FADEC to Fuel Metering Unit (FMU) lead Ch. A or B)
  • NH Sensor (Ch. A or B)
  • Fuel Metering Unit (FMU)
  • Full Authority Digital Electronic Control (FADEC)
