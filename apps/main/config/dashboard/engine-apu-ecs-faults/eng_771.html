Uptrim Discrete Comparison Between Channel A and B - (Engine Fault Code 771)

FIM TASK 73-20-00-810-839 (TR-FIM)

The fault isolation procedure that follows is to correct the Full Authority Digital Electronic Control (FADEC) fault codes. The registered fault code is 771.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Full Authority Digital Electronic Control (FADEC)
  • Opposite Engine Propeller Electronic Control (PEC)
  • Airframe Wiring Harness
