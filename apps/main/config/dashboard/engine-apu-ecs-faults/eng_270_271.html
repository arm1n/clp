PEC Feather Logic Fault (Status) - (Engine Fault Code 270/271)

FIM TASK 61-20-06-810-834 (TR-FIM)

This fault code is generated as a result of an apparent mismatch between manual feather not being selected and an unfeather demand. This is an internal PEC fault.

The fault isolation logic has determined that MANUAL FEATHER = Off (Not selected) and Unfeather demand = On (unfeather not selected).

The fault code is advisory but will not be annunciated to the flight crew as a PEC caution. This fault code is designated as no despatch for TLD purposes and therefore will also illuminate the powerplant message on the engine display.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • PEC
