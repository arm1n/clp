PROP CAUTION lamp drive fault - (Engine Fault Code 346/347)

FIM TASK 61-20-07-810-816 (TR-FIM)

This fault code is generated as a result of the propeller caution lamp driver circuit within the PEC having current apparently flowing, when there is no actual requirement being identified to indicate a fault.

The fault detection logic has determined that there is apparent current in the caution lamp circuit when the propeller caution demand is off.

The fault code is advisory but will not be annunciated to the flight crew as a PEC caution. This fault code is designated as no despatch for TLD purposes and therefore will also illuminate the powerplant message on the engine display.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Low battery voltage to the PEC
  • PEC
  • Wiring: PEC or airframe harness
  • CWP
  • PEC internal fault
  • CWP C/B pull
