Autofeather armed lamp failed - (Engine Fault Code 324/325)

FIM TASK 61-20-07-810-812 (TR-FIM)

This fault code is generated as a result of an apparent indication that the autofeather arm light has failed to illuminate upon demand

The fault isolation logic has determined that (AF armed lamp = off) and (AF lamp enable) and (AF state = AFFAI or AF state = AFARM).

This fault code will inhibit the autofeather advisory when PLA goes into the take−off range. There is no impact on TLD. This fault is typically annunciated during the take−off roll, which will result in a rejected take−off.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Caution and Warning panel
  • Fuel control wiring harness
  • Aircraft wiring
  • PEC
