No Dispatch Annunciation Flag - (Engine Fault Code 998 and 999)

FIM TASK 77-31-00-810-813 (TR-FIM)

These are EMU output flags which show that there is a NO DISPATCH condition and that the POWERPLANT caption has come into view on the Engine Display. Although there is no maintenance action necessary for these fault codes, there will be other fault codes stored in the FADEC or the PEC that have caused the NO DISPATCH condition.

Fault code 998 shows that there is a NO DISPATCH condition because of fault codes in the FADEC.

Fault code 999 shows that there is a NO DISPATCH condition because of fault codes in the PEC.
