Stack bound violation - (Engine Fault Code 98/99)

FIM TASK 61-20-02-810-831 (TR-FIM)

This fault code is generated as a result of key data within the PEC software having been overwritten. This is an internal PEC fault.

The fault isolation logic has determined that following the watchdog being serviced, the password set up on the stack has been overwritten

The fault code is advisory but will not be annunciated to the flight crew as a PEC caution. This fault code is designated as No Despatch for TLD purposes and therefore will also illuminate the powerplant message on the engine display.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • PEC
