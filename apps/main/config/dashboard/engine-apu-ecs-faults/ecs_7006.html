L ANA WING LEAK - (ECS Fault Code 7006)

CH 1 (LH) FIM TASK 36-10-00-810-801 (TR-FIM) CH 2 (RH) FIM TASK 36-11-34-810-802 (TR-FIM)
Possible Causes:
  • The CDS shows an L ANA WING LEAK status message when the environmental control system (ECS) electronic control unit (ECU) left analog channel senses a left−wing−spar bleed−duct leak.
  • The caution and warning panel (CAWP) #1 BLEED HOT or #2 BLEED HOT caution light can also come on.
The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):
  • Wing Spar Duct
