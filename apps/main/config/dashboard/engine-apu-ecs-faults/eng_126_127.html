Beta unreasonable - (Engine Fault Code 126/127)

FIM TASK 61-20-03-810-814 (TR-FIM)

This fault code is generated as a result of an apparent blade angle, that can only occur on the ground, being indicated during flight. This fault will only be detected when the propeller is rotating.

The fault accommodation logic has determined that the PCU beta feedback transducer (BFT) voltage output is indicating a propeller blade angle of less than 10 degrees in flight. This could be caused by electrical faults causing an offset in the measured blade angle (an in−range fault).

This fault code is designated as No Dispatch for time limited despatch (TLD) purposes and therefore will also illuminate the POWERPLANT advisory message on the engine display. If this fault is present on both channels or if the other PEC channel has fault code 120/121, 122/123, 126/127 or 130/131, the PEC caution light will be illuminated on the CWAS.

NOTE: If any combination of faults 120/121, 124/125, 126/127 or 130/131 are present on both PEC channels at the same time, the propeller will automatically feather on the ground if the PLA is moved below FLIGHT IDLE.

If a hard (non−intermittent) fault of the BFT signal is present when the propeller is not rotating, the affected PEC channel will also indicate fault code 134. Code 134 provides a real time status of the BFT signal validity and will clear by itself as soon as valid blade angle is detected by the affected PEC channel.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Loose or dirty connectors on BFT or PEC
  • Worn BFT connector pin and sockets
  • PEC wiring harness (wiring fault)
  • PCU (BFT fault)
  • PEC (internal fault)
