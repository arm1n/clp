Watchdog error counter increment failure - (Engine Fault Code 102/103)

FIM TASK 61-20-03-810-804 (TR-FIM)

This fault code is generated following a reset where the watchdog reset counter has not been incremented. This is an internal PEC fault.

The fault isolation logic has determined that following a reset the watchdog reset counter has not been incremented.

The fault code is advisory but will not be annunciated to the flight crew as a PEC caution. This fault code is designated as No Despatch for TLD purposes and therefore will also illuminate the powerplant message on the engine display.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • PEC
