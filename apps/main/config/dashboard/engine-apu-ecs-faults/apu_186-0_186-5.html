OIL DE−PRIME VLV SHORT ( Status ) - (APU Fault Code 186-0 to 186-5)

FIM TASK 49-90-00-810-801 (TR-FIM)

This task is for the centralized diagnostic system (CDS) fault message 186–0/2/3/4/5 OIL DE−PRIME VLV SHORT. This fault is set when the full authority digital electronic controller (FADEC) senses a short in the 28 V DC circuit to the deprime solenoid valve. This fault will not stop auxiliary power unit (APU) operation.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Deprime Solenoid Valve
