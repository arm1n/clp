Torque Fluctuation - (Engine Fault Code 1504)

FIM TASK 77-31-00-810-805 (TR-FIM)

This fault code occurs when torque fluctuation is detected.

Determine if any of the fault codes could show the cause of the torque fluctuation. Do the faultisolation in the relevant AMM section. If necessary, you can get details of the event if youdownload the trace recording from the EMU to the laptop computer which has the Ground Based Software
