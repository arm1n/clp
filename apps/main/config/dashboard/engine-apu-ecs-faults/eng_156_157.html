MPU 7P IN extra/missing pulses - (Engine Fault Code 156/157)

FIM TASK 61-20-04-810-809 (TR-FIM)

This fault code is generated as a result of an apparent loss of some of the MPU pulses from the master (LH) PEC to the slave (RH) PEC. This fault will result in the loss of synchrophase control, and should only be generated on the RH PEC.

The fault isolation logic has determined that there is a lack of phase signals on the MPU 7P IN signal.

The fault code is advisory and will not be annunciated to the flight crew. This fault code is designated as unlimited despatch for TLD purposes.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • LH powerplant MPU (with fault code 160)
  • LH powerplant PEC
  • RH powerplant PEC
  • Electrical Wiring
