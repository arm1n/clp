This fault code is generated as a result of an apparent deselection of an input used only for software overlay purposes. This is an internal PEC fault.

The fault isolation logic has determined that the software is trying to download or select overlay data.

The fault code is advisory but will not be annunciated to the flight crew as a PEC caution. This fault code is designated as no despatch for TLD purposes and therefore will also illuminate the powerplant message on the engine display.


PRE SB84-61-01

This fault code is generated as a result of an apparent mismatch in the PEC permissions required to unfeather the propeller in maintenance mode. This is an internal PEC fault.

The fault isolation logic has determined that (MANUAL UNFEATHER = on) and (MANUAL FEATHER = off) and (AF FET MON = off).

The fault code is advisory but will not be annunciated to the flight crew as a PEC caution. This fault code is designated as no despatch for TLD purposes and therefore will also illuminate the powerplant message on the engine display.
