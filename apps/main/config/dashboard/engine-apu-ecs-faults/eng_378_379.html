Beta not calibrated. lane not active long enough - (Engine Fault Code 378/379)

FIM TASK 61-20-08-810-821 (TR-FIM)

This fault code has been generated during beta calibration on the control lane is not in control for long enough for a calibration to be completed.

The fault isolation logic has determined that there has been insufficient time for the control lane to complete a calibration.

The fault code is advisory but will not be annunciated to the flight crew as a PEC caution. This fault code is designated as no despatch for TLD purposes and therefore will also illuminate the powerplant message on the engine display.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • PEC
