Invalid FADEC T1.8 - (Engine Fault Code 462/463)

FIM TASK 61-20-10-810-828 (TR-FIM)

This fault code is generated as a result of an apparent invalid total inlet temperature (T1.8) signal from the engine FADEC to the PEC. For Post Modsum 4W113558 this fault will only be annunciated when the propeller is rotating.

The fault isolation logic has determined that T1.8 is invalid as supplied via the RS422 databus from the FADEC.

The fault code is advisory and will not be annunciated to the flight crew. This fault code is designated as unlimited despatch for TLD purposes.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Total inlet temperature sensor
  • Control wiring harness (FADEC to T1.8 sensor)
  • FADEC
  • Fuel control electrical wiring harness
  • PEC
