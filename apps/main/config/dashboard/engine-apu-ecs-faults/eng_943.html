ECIU Low Fuel Pressure Lamp - (Engine Fault Code 943)

FIM TASK 73-20-00-810-885 (TR-FIM)

The fault isolation procedure that follows is to correct the full authority digital electronic control (FADEC) fault codes. The registered fault code is 943.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Airframe Wiring Harness
  • Caution and Warning Panel
  • Engine Cockpit Interface Unit (ECIU)
  • Full Authority Digital Electronic Control (FADEC)
