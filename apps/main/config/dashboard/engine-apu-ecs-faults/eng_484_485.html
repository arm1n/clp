Invalid FADEC ACT NOISE PRES bit (Status) - (Engine Fault Code 484/485)

FIM TASK 61-20-10-810-825 (TR-FIM)

This task is for the Propeller Electronic Control Unit (PEC) fault code 484 − Invalid FADEC ACT NOISE PRES bit (On aircraft without Modsum 4−113264) or Spare (On aircraft with Modsum 4−113264). During fault isolation if any corrective action is performed a PEC fault clear must be completed. This will make sure that the problem has been fixed and the fault code does not reappear.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):
