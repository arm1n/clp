Maximum Cruise Rating Signal - (Engine Fault Code 819 and/or 820)

FIM TASK 73-20-00-810-859 (TR-FIM)

The fault isolation procedure that follows is to correct the full authority digital electronic control (FADEC) fault codes. The registered fault codes are 819 and/or 820.

Fault code 819 is registered when the MCR switch on the engine control panel is pressed for more than 10 seconds.

Fault code 820 is registered when the FADEC receives a different switch position on channel A and on channel B, coming from the same MCR switch in the cockpit.

The switch is located on the fuel/ engine/ propeller control panel.

The switch transmits its signal to the Engine Cockpit Interface Unit (ECIU). Then, the ECIU relays the information to the FADEC using ARINC 429.

If fault code 819 is registered, it is possible that it was triggered because the switch was pressed for more than 10 seconds and there are no malfunctioning components.

NOTE: In maintenance mode, the fault is triggered if the switch is pressed for more than 15 seconds.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Engine Cockpit Interface Unit (ECIU)
  • Engine Control Panel
  • Airframe Wiring Harness
