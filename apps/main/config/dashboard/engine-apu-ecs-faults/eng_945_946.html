ECIU Oil Cooler Flap Output - (Engine Fault Code 945 and/or 946)

FIM TASK 73-20-00-810-887 (TR-FIM)

This fault is triggered when the feed back signal from the oil cooler flap door position is not in agreement with the command signal from the FADEC.

The fault is latched when disagreement is detected for more than 5.5 seconds.

NOTE: In Maintenance Mode, the fault is latched when disagreement is detected for more than 15 seconds.

The FADEC request a flap position to the engine cockpit interface unit (ECIU) using the ARINC 429 link.

Then the ECIU sends a command signal to the flap actuator to move the flap door to the requested position from the FADEC.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Air Cooled Oil Cooler Flap Door Actuator
  • Air Cooled Oil Cooler Flap Door
  • Engine Cockpit Interface Unit (ECIU)
  • Full Authority Digital Electronic Control (FADEC)
