FD OVR TMP 4 - (ECS Fault Code 7114)

CH 1 (LH) FIM TASK 21-60-00-810-873 (TR-FIM) CH 2 (RH) FIM TASK 21-60-00-810-873 (TR-FIM)
Possible Causes:
  • This fault isolation procedure is for when the central diagnostic system (CDS) shows the FD OVR TMP4 message, Fault Code 7114. This is a DUAL DUCT HOT Detection Monitor Fault.NOTE: The acronyms FD (flight deck) and FLT COMPT (flight compartment) are interchangeable.
  • The caution and warning panel (CAWP) FLT COMPT and CABIN DUCT HOT caution light will also come on.NOTE: The DUCT HOT lamps will remain illuminated until the condition no longer exists and the PACK Auto/Man/Off switches have been set to OFF.
  • The CDS shows this message when the system has determined that the FD zone supply over temperature switch is open (sensed duct temperture is greater than 190 ±10 °F (87.8 ±5.6 °C) and at least one of the dual FD pack outlet temperature sensors sensed a temperature greater than 190.0 °F (87.8 °C) for a minimum of 20 seconds. Fault Code 7114 can also set if either Fault Code 7111, Fault Code 7112, or Fault Code 7113 were set and stayed set for a minimum of 20 seconds after the FD turbine SOV was commanded closed. When this fault is set, both the FLT COMPT and CABIN DUCT HOT cautions lamps come on and the pack flow control valve is commanded closed to stop airflow to the ACMs and pack bypass valves.
The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):
  • ECU
  • Pack Flow Control and Shut−off Valve Filter
