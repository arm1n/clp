Other Lane Discrete I/F - (Engine Fault Code 6/7)

FIM TASK 61-20-01-810-802 (TR-FIM)

This fault code is generated as a result of an invalid value for the control priority discretes. This is an indication of an internal PEC fault.
The fault isolation logic has determined that the interlane discretes are invalid. The fault code is advisory but will not be annunciated to the flight crew as a PEC caution. This fault code is designated as No Despatch for TLD purposes and therefore will also illuminate the powerplant message on the engine display.



The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):
  • PEC
