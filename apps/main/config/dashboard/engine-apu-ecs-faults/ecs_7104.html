FD (CAB )ACM CMP OT1 - (ECS Fault Code 7104)

CH 1 (LH) FIM TASK 21-60-00-810-867 (TR-FIM) CH 2 (RH) FIM TASK 21-60-00-810-867 (TR-FIM)
Possible Causes:
  • This fault isolation procedure is for when the central diagnostic system (CDS) shows the FD ACM CMP OT1 or CAB ACM OT1 message, Fault Code 7104. This is a PACK HOT Detection Monitor Fault.NOTE: The acronyms FD (flight deck) and FLT COMPT (flight compartment) are interchangeable.NOTE: The acronym CAB is interchangeable with the term CABIN.
  • The caution and warning panel (CAWP) FLT COMPT or CABIN PACK HOT caution light will also come on.NOTE: The PACK HOT lamp will remain illuminated until the condition no longer exists and the PACK Auto/Man/Off switch has been selected OFF.
  • The CDS shows this message when the system has determined the FD or CABIN compressor outlet temperature sensor is above 430 °F (221 °C) and the compressor temperature switch is open (sensed temperature above switch point of 450 ±10 °F (232±5.6 °C) for 10 seconds). However, if the FD or CABIN compressor outlet temperature sensor is faulted (Fault Code 3701), Fault Code 7104 is set only by the compressor temperature switch going open for 10 seconds. When this fault code is set the FLT COMPT or CABIN PACK HOT caution lamp comes on.
