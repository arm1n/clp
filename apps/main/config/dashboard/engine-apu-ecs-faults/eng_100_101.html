Watchdog failure to reset processor - (Engine Fault Code 100/101)

FIM TASK 61-20-03-810-803 (TR-FIM)

This fault code is generated as a result of the PEC software failing to reset the processor immediately following the writing of a mistrigger command to the watchdog. This is an internal PEC fault.

The fault isolation logic has determined that following the writing of a mistrigger command to the watchdog, a watchdogâˆ’fail to reset fault has been set when the processor is not reset within 0.001 ms.

The fault code is advisory but will not be annunciated to the flight crew as a PEC caution. This fault code is designated as No Despatch for TLD purposes and therefore will also illuminate the powerplant message on the engine display.
