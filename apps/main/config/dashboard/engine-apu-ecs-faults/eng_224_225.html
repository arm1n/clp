The GBE switch stays in the off position - (Engine Fault Code 224/225)

FIM TASK 61-20-05-810-826 (TR-FIM)

This fault code is generated as a result of a discrepancy between the voltages being recorded on either side of the Ground Beta Enable switch internal to the PEC. This is not an indication of a Ground Beta Enable Solenoid problem on the PCU. The fault is an internal PEC problem.

The fault detection logic has determined that, on the active lane, PLA BETA = 1, GROUND BETA ENABLE SOL MON = 1 and GROUND BETA ENABLE SOL = Switch Closed (demanded closed) for a confirmation time of 2.5 seconds.

This fault code will be annunciated as a PEC caution on the ACWS. This fault code is designated as No Despatch for TLD purposes and therefore will also illuminate the powerplant message on the engine display.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • PEC
