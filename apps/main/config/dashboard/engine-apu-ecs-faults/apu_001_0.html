FADEC FAULT ( Status ) - (APU Fault Code 001-0)

FIM TASK 49-60-00-810-801 (TR-FIM)

The CDS shows the 001-0 FADEC FAULT message when the auxiliary power unit (APU) full authority digital electronic controller (FADEC) cannot read the state of the start switch discrete input.

The caution and warning panel (CAWP) APU and APU CONTROL panel FAIL caution lights can come on.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • APU FADEC
