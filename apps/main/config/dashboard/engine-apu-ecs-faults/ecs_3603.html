FD BYP VLV - (ECS Fault Code 3603)

CH 1 (LH) FIM TASK 21-60-00-810-853 (TR-FIM) CH 2 (RH) FIM TASK 21-60-00-810-853 (TR-FIM)
Possible Causes:
  • The CDS shows this message when there is a flight compartment pack bypass valve malfunction.
  • The caution and warning panel (CAWP) can show one or more of the related caution lights that follow on:FLT COMPT PACK HOT or FLT COMPT DUCT HOT
The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):
  • Pack Bypass Valve
  • Environmental Control System Electronic Control Unit (ECU)
