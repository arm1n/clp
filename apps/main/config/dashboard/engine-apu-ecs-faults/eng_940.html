FADEC Power Supply not available to PEC (Status) - (Engine Fault Code 940)

FIM TASK 73-20-00-810-890 (TR-FIM)

This fault is flagged when the FADEC detects an overcurrent condition on the power supply circuit to the PEC.

Only one of the two channels of the FADEC provides power to the PEC at a time.

During the powerâˆ’up of the FADEC and the initialization testing, the output current driver to the PEC is tested. Therefore, a FADEC fault code clear will indicate if the fault is still present, with no engine run necessary.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Control Wiring Harness
  • Full Authority Digital Electronic Control (FADEC)
  • Propeller Electronic Control (PEC)
