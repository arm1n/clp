I − BIT aborted: Lane change occured - (Engine Fault Code 74/75)


FIM TASK 61-20-02-810-837 (TR-FIM)

This fault code is generated as a result of the failure of the propeller system to successfully complete an autofeather test (Pre Modsum 4W113558), or an autofeather, OSG or automated flight fine stop test (Post Modsum 4W113558), resulting from the PEC changing lanes during the test. This can only be assumed to be as a result of a non−related problem arising during the test process.

The fault isolation logic has detected that the control lane within the PEC has changed from B to A, or A to B.

The fault code is advisory and will not be annunciated to the flight crew. This fault code is designated as unlimited despatch for TLD purposes. This fault code does NOT change the previous status of the autofeather test pass/fail and is a diagnostic fault code to show the AF test was aborted. This fault code only shows for 5 seconds and then self clears.

During fault isolation for OSG and AF test faults, the tests must be done again after corrective action has been taken.
The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):
  • Unrelated fault (in conjunction with other fault codes)
  • Local PEC fault
