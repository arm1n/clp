CAB TSOV - (ECS Fault Code 7109)

CH 1 (LH) FIM TASK 21-60-00-810-866 (TR-FIM) CH 2 (RH) FIM TASK 21-60-00-810-866 (TR-FIM)
Possible Causes:
  • This fault isolation procedure is for when the central diagnostic system (CDS) shows the CAB TSOV message, Fault Code 7109. This is a PACK HOT Detection Monitor Fault.NOTE: The acronym CAB is interchangeable with the term CABIN.
  • The caution and warning panel (CAWP) CABIN PACK HOT caution light will also come on.NOTE: The PACK HOT lamp will remain illuminated until the condition no longer exists and the PACK Auto/Man/Off switch has been selected OFF.
  • The CDS shows this message when the system has determined the compressor outlet and pack outlet temperatures do not change after shutting down the ACM.
