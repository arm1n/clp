FAIL TO LIGHT ( Status ) - (APU Fault Code 133-0 to 133-2)

FIM TASK 49-20-00-810-803 (TR-FIM)

Operational Test of the APU Igniter Plugs

AMM TASK 49-40-04-710-801 (TR-AMM)

The CDS shows this message when the auxiliary power unit (APU) full authority digital electronic controller (FADEC) senses that APU EGT does not increase after 3% speed during the start mode.

The caution and warning panel (CAWP) APU and APU CONTROL panel FAIL caution lights can come on.

NOTE: If one start fuel nozzle is blocked, or one igniter has malfunctioned, the APU should start with the other start fuel nozzle or igniter. This discrepancy may not be seen until two of the components malfunction. The start fuel nozzles and the igniters can be swapped as part of the troubleshooting process.
