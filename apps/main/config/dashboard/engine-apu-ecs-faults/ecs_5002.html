TMP DISC MISCMP - (ECS Fault Code 5002)

CH 1 (LH) FIM TASK 36-10-00-810-870 (TR-FIM) CH 2 (RH) FIM TASK 36-10-00-810-870 (TR-FIM)
Possible Causes:
  • The CDS shows this message when the aircraft configuration discrete inputs for the bleed temperature sensor option disagree. Fault Code 5002 is set on the channel that does not have the jumper wire installed to the 2100−P2 or 2100−P5 connector on the ECU.
  • The caution and warning panel (CAWP) #1 BLEED HOT or #2 BLEED HOT caution lamp come on whenever the TMP DISC MISCMP message, Fault Code 5002 is active, the aircraft is on the ground, and both engines are OFF. This is done to alert maintenance that the jumper wire needs to be installed before there is a loss of protection for a bleed over temperature event. Once the engine is started the BLEED HOT lamp extinguishes even if the CDS fault has not been cleared.
