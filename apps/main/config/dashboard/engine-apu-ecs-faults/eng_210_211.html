OSG 1−Bit failure (Status) - (Engine Fault Code 210/211)

FIM TASK 61-20-05-810-830 (TR-FIM)

This task is for fault code 210 − OSG I−Bit failure (On aircraft without Modsum 4−113264) or OSG governing fault (On aircraft with Modsum 4−113264). This fault code is generated as a result of the propeller Np increasing above 940 rpm during the OSG test. This is an indication of a hydro−mechanical problem with the OSG.

The fault isolation logic has determined that during the OSG test the propeller Np speed has increased above 940 rpm.

This fault code is advisory and will not be annunciated to the flight crew. This fault code is designated as unlimited despatch for TLD purposes.

During fault isolation for overspeed governor (OSG) test faults, the tests must be completed again after corrective action has been taken. If the tests pass then a fault clear will be necessary to clear the fault in question.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Overspeed governor
