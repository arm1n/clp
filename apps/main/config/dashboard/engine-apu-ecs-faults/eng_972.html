P2.2 Bleed Valve Health Flag - (Engine Fault Code 972)

FIM TASK 73-20-00-810-828 (TR-FIM)

The fault isolation procedure that follows is to correct the full authority digital electronic control (FADEC) fault codes. The registered fault code is 972.

If the fault code occurs in one or both channels clear the fault code. Return the aircraft to service (Refer toAMM TASK 77−31−00−710−803 ).

NOTE: No maintenance is necessary and no component needs to be replaced.
