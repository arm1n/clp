P3 Bleed Precooler Input Comparision Between Channel A and B - (Engine Fault Code 768)

FIM TASK 73-20-00-810-838 (TR-FIM)

The fault isolation procedure that follows is to correct the full authority digital electronic control (FADEC) fault codes. The registered fault code is 768.

The FADEC received the data on the position of the P2.2 Shut−Off Valve (SOV). The two possible positions are ON or OFF.

This fault code indicates a difference between the data received from the shut−off valve on channel A of the FADEC and the data received on channel B from the same valve.

The fault is latched when the disagreement is read by the FADEC for more than 15 seconds. NOTE: An engine run is required to repeat the fault.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • P2.2 shut−off valve
  • Airframe wiring harness
  • FADEC
