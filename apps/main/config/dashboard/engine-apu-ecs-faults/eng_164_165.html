NQa inconsistent with NP and NQb - (Engine Fault Code 164/165)

FIM TASK 61-20-04-810-822 (TR-FIM)

This fault code is generated as a result of an apparent difference between the torque shaft speed signal Qa, and the propeller Np MPU speed signal and the torque shaft speed signal Qb. This fault code results in the loss of the autofeather function.

The fault accommodation logic has determined that Qa differs by more than 25 prpm from Np and Qb.
