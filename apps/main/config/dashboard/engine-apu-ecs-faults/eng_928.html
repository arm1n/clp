P3 Signal Not Valid When Engine Runs - (Engine Fault Code 928)

FIM TASK 73-20-00-810-874 (TR-FIM)

The fault isolation procedure that follows is to correct the full authority digital electronic control (FADEC) fault codes. The registered fault code is 928.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • P3 Transducer Air Tube
  • P3 Transducer
  • Full Authority Digital Electronic Control (FADEC)
