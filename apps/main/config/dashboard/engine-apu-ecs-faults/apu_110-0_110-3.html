SPEED AT POWER UP ( Status ) - (APU Fault Code 110-0 to 10-3)

FIM TASK 49-60-00-810-850 (TR-FIM)

The CDS shows the 110-3 SPEED AT POWER UP message when, during power up, the auxiliary power unit (APU) full authority digital electronic controller (FADEC) senses an engine speed more than 25% or has a loss of input power.

NOTE: It is possible that you will see the 110-3 SPEED AT POWER UP message if the APU FADEC senses a power loss during the shutdown mode.

The caution and warning panel (CAWP) APU and APU CONTROL panel FAIL caution lights can come on.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • APU FADEC
