The GBE switch stays in the off position - (Engine Fault Code 222/223)

FIM TASK 61-20-05-810-827 (TR-FIM)

This fault code is generated as a result of an apparent discrepancy between the voltages across the ground beta enable solenoid, as measured within the PEC.

The fault detection logic has determined that, on the active lane, PLA BETA = 1, GROUND BETA ENABLE SOL MON = 0, and GROUND BETA ENABLE SOL = SWITCH OPEN (demanded open).

This fault code will be annunciated as a PEC caution on the ACWS. This fault code is designated as No Despatch for TLD purposes and therefore will also illuminate the powerplant message on the engine display.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • PEC WIRING
  • PEC
  • PCU
