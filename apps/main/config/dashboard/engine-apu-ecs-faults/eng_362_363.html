Autofeather PSU - (Engine Fault Code 362/363)

FIM TASK 61-20-08-810-803 (TR-FIM)

This fault code is generated as a result of an apparent loss of the autofeather power supply.

The fault detection logic has determined that AF POWER OK = 0. This fault code will be confirmed after 70 seconds when PMA power is not available, or after 8 seconds when the PMA power is available.

  • PMA
  • FADEC
  • Fuel control electrical wiring harness
  • PEC
