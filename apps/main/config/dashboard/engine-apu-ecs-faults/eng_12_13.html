Other lane. Parity Error - (Engine Fault Code 12/13)

FIM TASK 61-20-01-810-805 (TR-FIM)

This fault code is generated as a result of an apparent error in the FADEC to PEC RS422 data, supplied to the PEC lane that is not in control. This is indicative of an internal PEC fault. The fault isolation logic has determined that the data on the FADEC-PEC RS422 databus may be invalid. The fault code is advisory and will not be annunciated to the flight crew. This fault code is designated as unlimited despatch for TLD purposes.



The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):
  • PEC
