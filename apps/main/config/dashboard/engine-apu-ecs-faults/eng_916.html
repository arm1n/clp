Low Oil Pressure Detected - (Engine Fault Code 916)

FIM TASK 73-20-00-810-871 (TR-FIM)

FIM TASK 79-30-00-810-801 (TR-FIM)

This fault isolation procedure that follows is for an engine that has low or a loss of oil pressure.

The following are a list of possible malfunctioned systems and/or line replaceable units (LRU): - Replaced Line Replaceable Units (LRU) - Underservicing of the Oil System - External Oil Leak - Oil Pressure Transducer - Low Oil Pressure Switch - Scavenge Filter - Fuel Heater - Oil Pressure Regulation - Pressure/Scavenge Pump Assembly - Main Oil Filter
