Dummy load fault - (Engine Fault Code 262/263)

FIM TASK 61-20-06-810-819 (TR-FIM)

This fault code is generated as a result of the apparent inability of the standby lane to output the required current to the Pitch Control Unit servo valve when using a dummy load within the PEC. This is an internal PEC fault.

The fault isolation logic has determined that the standby lane drive current to the PCU servo valve is not within 3mA of the required value when using a dummy load within the PEC.

This fault code will be annunciated as a PEC caution on the ACWS. This fault code is designated as no despatch for TLD purposes and therefore will also illuminate the powerplant message on the engine display.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • PEC
