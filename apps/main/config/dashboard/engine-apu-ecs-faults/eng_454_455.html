Invalid FADEC REDUCED NP Requested - (Engine Fault Code 454/455)

FIM TASK 61-20-10-810-803 (TR-FIM)

This fault code is generated as a result of an apparent loss of the local powerplant PEC s ability to enter the low rpm approach mode. For Post Modsum 4W113558, the fault code will only be annunciated when the propeller is rotating.

The fault detection logic has determined that the RS422 databus input from the local FADEC is indicating that reduced NP is not available.

The fault code is advisory and will not be annunciated to the flight crew. This fault code is designated as unlimited despatch for TLD purposes.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Fuel control electrical wiring harness (local or remote powerplant)
  • Local FADEC
  • Local PEC
