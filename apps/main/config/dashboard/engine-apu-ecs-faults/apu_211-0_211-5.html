RTL OUTPUT SHORT ( Status ) - (APU Fault Code 211-0 to 211-5)

FIM TASK 49-10-00-810-805 (TR-FIM)

The fault isolation procedure that follows is for the RTL OUTPUT SHORT message on the centralized diagnostics system (CDS). This message is the result of the following condition:

The full authority digital electronic controller (FADEC) senses a short in the ready−to−load (RTL) signal.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • APU FADEC
