Spool Down Trend Alert (NH) - (Engine Fault Code 1700)

FIM TASK 77-31-00-810-806 (TR-FIM)

The Fault Isolation procedure that follows is to correct the engine monitoring unit (EMU) fault code. The registered fault code is 1700. This fault code occurs when the trend for the spool down time for the NH rotor is less than normal.

Engine #1 (#2) Spooldown alert

NOTE: If necessary, you can get details of the event if you download the trace recording from the EMU to the laptop computer which has the Ground Based Software
