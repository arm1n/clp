PACK CMP OT1 - (ECS Fault Code 7107)

CH 1 (LH) FIM TASK 21-60-00-810-875 (TR-FIM) CH 2 (RH) FIM TASK 21-60-00-810-875 (TR-FIM)
Possible Causes:
  • This fault isolation procedure is for when the central diagnostic system (CDS) shows the PACK CMP OT1 message, Fault Code 7107 on either the left or right channel. This is a DUAL PACK HOT Detection Monitor Fault.NOTE: The acronyms FD (flight deck) and FLT COMPT (flight compartment) are interchangeable.NOTE: The acronym CAB is interchangeable with the term CABIN.
  • The caution and warning panel (CAWP) FLT COMPT and CABIN PACK HOT caution light will also come on.NOTE: Both PACK HOT lamps will remain illuminated until the condition no longer exists and the PACK Auto/Man/Off switches have been selected OFF.
  • The CDS shows this message when the system has determined the compressor temperature switch is open (sensed temperature is above the switch point of 450 ±10 °F (232 ±5.6 °C ) and the FD or CABIN ACM cannot be controlled by the other ECU channel. This condition must exist for 10 seconds. When this fault code is set the FLT COMPT and CABIN PACK HOT caution lamps come on.
