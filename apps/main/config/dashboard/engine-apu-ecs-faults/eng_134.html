Beta feedback not available (propeller stationary ) - (Engine Fault Code 134)

FIM TASK 61-20-03-810-822 (TR-FIM)

This fault code is generated as a result of the apparent loss of the blade angle feedback signal from the Beta Feedback Transducer (BFT) located on the PCU, on the active or standby PEC control lane, when the propeller is not rotating. If PEC has multiple BFT related faults, Fault 134 should be rectified first.

The fault detection logic has determined that there is no valid value of propeller blade angle (Beta) on the active or standby PEC lane (channel) for a period longer than 2 seconds.

This fault code will clear by itself as soon as valid blade angle is detected by the PEC and it does not require fault clear.

The fault detection logic for fault code 134 is identical to the fault detection logic for fault code 130. However, fault code 130 can only occur with propeller rotating and does not clear by itself (it requires a fault clear).

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Loose or dirty connectors on BFT or PEC
  • Worn BFT connector pin and sockets
  • PEC wiring harness (wiring fault)
  • PCU (BFT fault)
  • PEC (internal fault)
