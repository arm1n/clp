Turbomachinery Chip Detector - (Engine Fault Code 938)

FIM TASK 73-20-00-810-882 (TR-FIM)

The fault isolation procedure that follows is to correct the full authority digital electronic control (FADEC) fault codes. The registered fault code is 938 (Refer to AMM79−22−00−750−801).

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Turbomachinery Chip Detector
  • Instrumentation Wiring Harness (A/C chip detector lead)
  • Airframe Wiring Harness
