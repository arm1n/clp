Invalid FADEC QgTrim - (Engine Fault Code 468/487)

FIM TASK 61-20-10-810-811 (TR-FIM)

This fault code is generated as a result of an apparent invalid FADEC Qg trim signal from the engine FADEC to the PEC. Qg trim is a signal representing engine torque shaft gain. For Post Modsum 4W113558, the fault code will only be annunciated when the propeller is rotating.

The fault isolation logic has determined that FADEC Qg trim is invalid, as supplied by the RS422 databus from the FADEC.

This fault code is advisory and will not be annunciated to the flight crew. This fault code is designated as unlimited despatch for TLD purposes.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • FADEC characterization plug
  • Control wiring harness (FADEC to torque sensor lead)
  • FADEC
  • Fuel control electrical wiring harness
  • PEC
