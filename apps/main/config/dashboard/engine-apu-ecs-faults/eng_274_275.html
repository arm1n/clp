Post SB84-61-01-A PEC Maintenance mode mismatch / - (Engine Fault Code 274-275)

Post SB84-61-01-A PEC FIM TASK 61-20-06-810-826 (TR-FIM)

Post SB84-61-01-A PEC:

This fault code is generated as a result of a difference in the maintenance mode status of the PEC and FADEC.

The fault isolation logic has determined that PEC MAINT disagrees with FADEC MAINT for more than 2 seconds. For Post Modsum 4W113558, this fault code will not be annunciated when the propeller is rotating.

This fault code is advisory and will not be annunciated to the flight crew. This fault code is designated as unlimited despatch for TLD purposes.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Maintenance discrete switch
  • Aircraft wiring
  • PEC
  • Fuel Control electrical wiring harness
  • FADEC
