BACK−UP OVERSPEED ( Status ) - (APU Fault Code 123-0 to 123-4)

FIM TASK 49-20-00-810-811 (TR-FIM)

The CDS shows a 123−0, 123−1,123−2,123−3, 0R 123−4BACK−UP OVERSPEED message when the auxiliary power unit (APU) full authority digital electronic controller (FADEC) senses that the APU speed is more than 108%.

The caution and warning panel (CAWP) APU and APU CONTROL panel FAIL caution lights can come on during the power up, watch, start, run or shutdown.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • APU FADEC
  • Speed Sensor
