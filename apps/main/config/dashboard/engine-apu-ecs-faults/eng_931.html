RGB Scavenge Filter Impending Bypass Switch - (Engine Fault Code 931)

FIM TASK 73-20-00-810-877 (TR-FIM)

The fault isolation procedure that follows is to correct the full authority digital electronic control (FADEC) fault codes. The registered fault code is 931.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Airframe Wiring Harness
  • Instrumentation Wiring Harness (scavenge oil filter impending bypass switch lead)
  • Scavenge Oil Filter Impending Bypass Switch
  • Full Authority Digital Electronic Control (FADEC)
