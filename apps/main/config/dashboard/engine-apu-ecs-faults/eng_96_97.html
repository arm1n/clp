Illegal overlay parameter - (Engine Fault Code 96/97)

FIM TASK 61-20-02-810-830 (TR-FIM)

This fault code is generated as a result of an illegal overlay parameter being used by the PEC software. This is an internal PEC fault.

The fault isolation logic has determined that when overlaying is not enabled, the software has found that the overlayable parameter values are not the same as the default values.

The fault code is advisory but will not be annunciated to the flight crew as a PEC caution. This fault code is designated as No Despatch for TLD purposes and therefore will also illuminate the powerplant message on the engine display.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • PEC
