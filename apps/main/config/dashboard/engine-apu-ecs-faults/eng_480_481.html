Invalid FADEC RIGGING TRIM bit - (Engine Fault Code 480/481)

FIM TASK 61-20-10-810-817. (TR-FIM)

This fault code is generated as a result of an apparent indication from the FADEC to the PEC that FADEC RIGGING TRIM is invalid. The signal is a permissive for various rigging operations and if it is invalid it may prevent certain maintenance activities from being successfully completed. For Post Modsum 4W113558, the fault code will only be annunciated when the propeller is rotating.

The fault isolation has determined that FADEC RIGGING TRIM is invalid, as supplied by the RS422 databus from the FADEC to the PEC.

The fault code is advisory and will not be annunciated to the flight crew. This fault code is designated as unlimited despatch for TLD purposes.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • ECIU
  • Airframe wiring
  • FADEC
  • Fuel control electrical wiring harness
  • PEC
