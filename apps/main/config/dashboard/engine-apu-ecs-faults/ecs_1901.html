L (R) PRCLR TMP SW - (ECS Fault Code 1901)

CH 1 (LH) FIM TASK 36-10-00-810-810 (TR-FIM) CH 2 (RH) FIM TASK 36-10-00-810-810 (TR-FIM)
Possible Causes:
  • The CDS shows this message when the L(R) PRCLR PR SW has high resistance.
  • The caution and warning panel (CAWP) #1 BLEED HOT or #2 BLEED HOT caution light can also come on.
  • Unserviceable Nacelle Shutoff Valve
  • Unserviceable Left or Right Precooler Temperature Switch
The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):
  • Precooler Temperature Switch
