Beta out of range on standby lane - (Engine Fault Code 122/123)

FIM TASK 61-20-03-810-818 (TR-FIM)

This fault code is generated as a result of an apparent error in the input or output voltage levels from the PCU beta feedback transducer (BFT), on the standby PEC control lane. This fault will only be detected when the propeller is rotating.

The fault accommodation logic has determined that either the excitation voltages supplied to the BFT by the PEC are outside of the normal range, or that the BFT output voltages supplied to the PEC by the BFT are outside of the normal range.

This fault code is designated as No Despatch for time limited despatch (TLD) purposes and therefore will also illuminate the POWERPLANT advisory message on the engine display. If this fault is present on both channels or if the other channel of the PEC has fault code 120/121, 124/125, 126/127 or 130/131, the PEC caution light will be illuminated on the CWAS.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Loose or dirty connectors on BFT or PEC
  • Worn BFT connector pin and sockets
  • PEC wiring harness (wiring fault)
  • BFT fault
  • PEC (internal fault)
