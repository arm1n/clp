ECU CHAN 1 (CHAN 2) - (ECS Fault Code 420D)

CH 1 (LH) FIM TASK 21-60-00-810-844 (TR-FIM) CH 2 (RH) FIM TASK 21-60-00-810-844 (TR-FIM)
Possible Causes:
  • The ECU monitors the Channel 1 and Channel 2 discrete signals to make sure the combination of discrete signals is correct. The correct combination of discrete signals is one signal grounded and one signal opened. If a different combination occurs, a fault message for Channel 1 or Channel 2 will show.
The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):
  • Environmental Control System Electronic Control Unit (ECU)
