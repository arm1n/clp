OVERTEMP AT START ( Status ) - (APU Fault Code 131-0 to 131-4)

FIM TASK 49-20-00-810-807 (TR-FIM)

Borescope Inspection of the Auxiliary Power Unit (APU)

AMM TASK 49-00-00-290-801 (TR-AMM)

The CDS shows a 131−4 OVERTEMP AT START message when the auxiliary power unit (APU) full authority digital electronic controller (FADEC) senses that the EGT is more than 816°C (1500°F) during the start mode.

The caution and warning panel (CAWP) APU and APU CONTROL panel FAIL caution lights can come on.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Engine Control Harness Assembly
  • APU FADEC
