Common Torque Motor Supply (Status) - (Engine Fault Code 967)

FIM TASK 73-20-00-810-893 (TR-FIM)

This fault code is triggered when there is a problem with the common power supply for the LRU torque motors.

There are three LRUs that have a common power supply to their torque motors: P2.2 Interstage Bleed Valve, P2.7 Intercompressor Bleed Valve and Fuel Metering Unit (FMU).

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • P2.2 Interstage Bleed Valve
  • P2.7 Intercompressor Bleed Valve
  • Fuel Metering Unit (FMU)
  • Control Wiring Harness
  • Full Authority Digital Electronic Control (FADEC)
