FADEC FAULT ( Status ) - (APU Fault Code 162-0 to 162-4)

FIM TASK 49-60-00-810-811 (TR-FIM)

The CDS shows the 162-4 FADEC FAULT message when the auxiliary power unit (APU) full authority digital electronic controller (FADEC) senses an internal hot switch stim failure condition during the power up mode.

The caution and warning panel (CAWP) APU and APU CONTROL panel FAIL caution lights can come on.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • APU FADEC
