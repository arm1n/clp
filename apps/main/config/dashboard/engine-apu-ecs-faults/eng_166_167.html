NQb inconsistent with NP and NQa - (Engine Fault Code 166/167)

FIM TASK 61-20-04-810-823 (TR-FIM)

This fault code is generated as a result of an apparent difference between torque shaft speed signal Qb, and propeller Np MPU speed signal and torque shaft speed signal Qa. This fault results in the loss of autofeather function.

The fault accommodation logic has determined that Qb differs by more than 25 prpm from Np and Qa.
