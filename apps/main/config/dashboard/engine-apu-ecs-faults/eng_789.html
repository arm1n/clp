P3 Signal Related Faults - (Engine Fault Code 740, 789 and/or 790)

FIM TASK 73-20-00-810-812 (TR-FIM)

The fault isolation procedure that follows is to correct the full authority digital electronic control (FADEC) fault codes. The registered fault codes are 740, 789 and/or 790.

These fault codes indicate a loss or a low signal from the P3 transducer to the FADEC.

Fault codes 740 and 789 indicate a disrupted signal.

Fault Code 790 indicates a difference between channel A and channel B.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • P3 Pressure Transducer
  • Control Wiring Harness (FADEC to P3 pressure Transducer lead)
  • Full Authority Digital Electronic Control (FADEC)
