ADU Pressure Comparison Between Channel A and B - (Engine Fault Code 816, 822 and/or 823)

FIM TASK 73-20-00-810-857 (TR-FIM)

The fault isolation procedure that follows is to correct the full authority digital electronic control (FADEC) fault codes. The registered fault codes are 816, 822 and/or 823.

The list that follows shows the possible malfunctioned systems and/or line replaceable units (LRU):

  • Static Air Temperature (SAT)
  • Static Air Pressure (SAP)
  • RAM Air Pressure
  • Air Data Unit (ADU)
  • Airframe Wiring Harness
