TOUCHED RUNWAY FIM TASK 31-53-00-810-801 (TR-FIM)
This fault isolation procedure is for when the TOUCHED RUNWAY warning light on the caution and warning panel (CAWP) is on.

The CAWP can show the TOUCHED RUNWAY warning light on when it senses that the aft-tail fuselage has struck the runway.
Functional test of the tail strike warning system
