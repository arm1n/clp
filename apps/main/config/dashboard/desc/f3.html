STBY BATTERY

FIM TASK 24-30-00-810-808 (TR-FIM)

The STBY BATTERY caution light comes on when the standby battery is not connected to left main feeder bus.

The Electrical Power Control Unit (EPCU) Continuous Built-In Test (CBIT) can show the fault code 29, K25 drive fail.

NOTE: The Central Diagnostic System (CDS) does not show any related status messages.

The Multi-Function Display (MFD) electrical system page and the caution and warning lights system show battery system indications. The Electrical Power Control Unit (EPCU) supplies indication data through the two Input/Output Processors (IOP1, IOP2) located in the Integrated Flight Cabinets (IFC) to the Electronic Instruments System (EIS).
