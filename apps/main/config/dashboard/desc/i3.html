#2 DC GEN FIM TASK 24-30-00-810-813 (TR-FIM)
The #2 DC GEN caution light on the CAWP comes on when the #2 DC generator is not connected to the left main feeder bus because of the source fault condition.
The #2 DC GEN caution light on the CAWP will also come on for the conditions which are not malfunctions that follow:
  • The DC generator toggle switch is set to OFF
  • External DC power is energizing the left main feeder bus

The Electrical Power Control Unit (EPCU) can show the related fault codes that follow:
  • 02, #2 DC generator feeder (Status)
    • FIM TASK 24-31-00-810-802 (TR-FIM)

  • 07, #2 DC generator (Status)
    • FIM TASK 24-31-01-810-802 (TR-FIM)

  • 11, K2 relay failed open (Status)
    • FIM TASK 24-31-31-810-802 (TR-FIM)
