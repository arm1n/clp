#1 DC GEN


FIM TASK 24-30-00-810-812 (TR-FIM)

The #1 DC GEN caution light on the CAWP comes on when the #1 DC generator is not connected to the left main feeder bus because of the source fault condition.
The #1 DC GEN caution light on the CAWP will also come on for the conditions which are not malfunctions that follow:
  • The DC generator toggle switch is set to OFF
  • External DC power is energizing the left main feeder bus

The Electrical Power Control Unit (EPCU) can show the related fault codes that follow:
  • 01, #1 DC generator feeder (Status)
    • FIM TASK 24-31-00-810-801 (TR-FIM)

  • 05, 05, #1 DC generator (Status)
    • FIM TASK 24-31-01-810-801 (TR-FIM)

  • 10, K1 relay failed open (Status)
    • FIM TASK 24-31-31-810-801 (TR-FIM)

  • SHOW ALL EPCU PAULT CODES
