FLAP DRIVE FIM TASK 27-50-00-810-801 (TR-FIM)
The CAWP FLAP DRIVE caution light comes on when the Flap Control Unit (FCU) senses a condition that follows:
  • Hydraulic pressure is sensed in the FPU with no flap selection (the shut−off valve stays open after the flap movement is completed).
  • The flap panels have stopped moving (stalled) before reaching the set position (a difference between the Flap Selected Position and the actual Flap Position feedback signal).
    NOTE: In this case, the FLAP DRIVE caution light will extinguish while the flaps are in motion.

The FCU can malfunction if:
  • Internal Electrically Erasable Programmable Read Only Memory (EEPROM) failure (CDS Reporting Function)
  • ARINC 429 transmission failure (CDS Reporting Function)
  • Flap position selected monitor (FDR Parameter).

NOTE: The flap system will continue to operate in a degraded mode when the FLAP DRIVE caution light is on, with the exception of a flap stall condition.
