Intermittent Display of ICE DETECTED Advisory Message (ED)


FIM TASK 30-80-00-810-806 (TR-FIM)

This Fault Isolation procedure is applicable when the engine display (ED) ICE DETECTED advisory message comes on for less than 60 + / - 5 seconds.



The central diagnostic system (CDS) can show the related messages as follows:

L ICE DETECTOR FAIL FIM TASK 30-80-00-810-802 (TR-FIM) R ICE DETECTOR FAIL FIM TASK 30-80-00-810-803 (TR-FIM)
