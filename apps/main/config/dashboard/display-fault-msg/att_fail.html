ATT & HDG FAIL on MFD
This fault isolation procedure is for when multifunction display (MFD) shows an ATT FAIL message. MFD shows an ATT FAIL message when it is set to show a primary flight display (PFD) reversion and does not receive valid pitch or roll data from the set attitude and heading reference unit (AHRU).

ATT FAIL ON PFD
This fault isolation procedure is for when primary flight display shows an ATT FAIL message. PFD shows an ATT FAIL message when it does not receive valid pitch or roll data from the set attitude and heading reference unit (AHRU).
