PITCH MISMATCH message on PFD

FIM TASK 34-20-00-810-803 (TR-FIM)

The FMA show a PITCH MISMATCH message when the flight data processing system (FDPS) senses an attitude and heading reference unit 1 (AHRU1) and AHRU2 pitch difference that is more than three degrees. The PFD electronic attitude director indicator (EADI) will show a PITCH mismatch flag.

The flight mode annunciator (FMA) in the primary flight display (PFD) can show a PITCH MISMATCH message

NOTE: The FMAs will show only the most important mismatch message and this is the primary mismatch indication.

NOTE: The attitude and heading reference system (AHRS) data must be valid and the electronic flight instrument system (EFIS) ATT/HDG SOURCE reversion switch must be set to the NORM position.








NOTE:

Reset AHRU 1 and AHRU 2 on Avionics CB Panel

  • AHRU 2 (A3) & (A10)
