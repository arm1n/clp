R FD FAIL, MFD 1/2 (Caution)

The Flight Mode Annunciator (FMA) shows the R FD FAIL message in the MFD1/2 for one of the conditions that follow:

  • The MFD1/2 does not receive a valid Flight Director (FD) data from the Flight Guidance Module 2 (FGM2) when it is set to the PFD.
  • The Input/Output Processor 1 (IOP1) does not receive a valid status from the FGM2.


R FD FAIL, PFD 1/2 (Caution)

The FMA shows the R FD FAIL message for one of the conditions that follow:

  • PFD1/2 does not receive a valid Flight Director (FD) data from the Flight Guidance Module 2 (FGM2)
  • Input/Output Processor module 1 (IOP1) does not receive a valid status from FGM2.




NOTE:

Reset FGM 1 and FGM 2 on Avionics CB Panel

  • (A4) Reset FGM 1 (F7) circuit breaker on Avionics CB panel
  • (A4) Reset FGM 2 (F6) circuit breaker on Avionics CB panel
