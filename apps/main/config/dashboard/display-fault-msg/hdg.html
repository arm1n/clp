HDG, MFD, navigation page (Caution)

The HDG mismatch flag is shown when the flight data processing system (FDPS) senses an attitude and heading reference system 1 (AHRS1) and AHRS2 heading difference of more than eight degrees.
The primary flight display (PFD) electronic horizontal situation indicator (EHSI) will also show a heading mismatch flag.
The PFD flight mode annunciator (FMA) can also show a HEADING MISMATCH message.

NOTE: The FMAs will show only the most important mismatch message and this is the primary mismatch indication.
