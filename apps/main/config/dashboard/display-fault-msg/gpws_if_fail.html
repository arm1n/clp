GPWS I/F FAIL




FIM TASK 31-41-00-810-870 (TR-FIM)

The ED shows this message when the input/output processor 1 (IOP1) senses that the ground proximity warning converter (GPWC) digital-to-analog function in the stall protection module 1 (SPM1) is supplying invalid data to the ground proximity warning system computer (GPWSC).

NOTE: This indication is shown on the ground only.

Message appears after power-up of displays.
GPWS SYST and ANN CB’s (A1 and B1 – Avionics) - Reset Simultaneously

IF display remains in self-test: Maintenance action is required prior to flight.
