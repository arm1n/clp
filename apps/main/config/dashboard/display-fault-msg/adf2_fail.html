ADF2 FAIL MESSAGE on ARCDU1/2

This fault isolation procedure is for when the automatic direction finder 2 (ADF2) display area of audio and radio control display unit 1 (ARCDU1) shows a FAIL message.

The ADF1 display area shows a FAIL message when ARCDU1/2 does not receive valid data from ADF1 through the flight data processing system (FDPS).




NOTE:

  • Reset following circuit breakers on AVIONICS CB PANEL:
    • H3 ADF 1
    H4 ADF 1

    F7 IOM/IOP/FGM1
    F6 IOM/IOP/FGM2

  • Cycle power to ARCDUs
