RA on PFD

FIM TASK 34-40-00-810-802 (TR-FIM)

The RA mismatch flag is shown when the flight data processing system (FDPS) senses a radio altimeter 1 (RA1) and RA2 altitude difference that is more than 20 feet. The PFD flight mode annunciator (FMA) can also show a RAD ALT MISMATCH message.

NOTE: The FMAs will only show the most important mismatch message and this is the primary mismatch indication.
