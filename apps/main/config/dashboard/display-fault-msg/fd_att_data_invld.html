FD ATT DATA INVLD on PFDs


FIM TASK 22-11-00-810-838 (TR-FIM)

This fault isolation procedure is for when the flight mode annunciator (FMA) in the primary flight display (PFD) show the FD ATT DATA INVLD message.

The FMA shows an FD ATT DATA INVLD indication for a flight director (FD) selection if the set attitude and heading reference system (AHRS) does not supply valid attitude data to the related flight guidance module (FGM).
