AFCS CONTROLLER INOP

FIM TASK 22-11-00-810-821 (TR-FIM)

This fault isolation procedure is for when the flight mode annunciator (FMA) in the primary flight display (PFD) shows the AFCS CONTROLLER INOP message.

NOTE: If the AP switch is stuck, it continues to set the mode. If any other switch is stuck, its operation is prevented.
