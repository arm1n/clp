L FD FAIL ON MFD 1 /2

The FMA shows the L FD FAIL message in MFD1/2 for one of the conditions that follow:

  • MFD1/2 does not receive a valid Flight Director (FD) data from the Flight Guidance Module 1 (FGM1) when it is set to Primary Flight Display (PFD)
  • Input/Output Processor Module 1 (IOP1) does not receive a valid status from FGM1.

L FD FAIL, PFD 1/2

The FMA shows the L FD FAIL message in PFD1/2 for one of the conditions that follow:

  • PFD1/2 does not receive a valid Flight Director (FD) data from the Flight Guidance Module 1 (FGM1)
  • Input/Output Processor module 1 (IOP1) does not receive a valid status from FGM1.




NOTE:
  • Reset FGM 1 (F7) circuit breaker on avionics CB panel

  • Reset FGM 2 (F6) circuit breaker on avionics CB panel
