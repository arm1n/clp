WX FAIL


WX FAIL on MFD1 FIM TASK 34-41-00-810-807 (TR-FIM) WX FAIL on MFD2 FIM TASK 34-41-00-810-808 (TR-FIM)

This fault isolation procedure is for when multifunction display 1 (MFD1) shows a WX FAIL message.

MFD1 shows a WX FAIL message when it does not receive valid weather radar data. The WX FAIL message is shown in the conditions that follow:

  • Defective weather radar receiver/transmitter
  • Defective weather radar antenna
  • Defective weather radar control panel
  • Defective antenna receiver/transmitter (ART) A453 data bus
  • Range discrepancy

ON both MFDs

  • Switch WXR OFF and then to TEST on WXR control panel
  • WAIT should be displayed in WHITE on left upper corner of MFD
  • after approx. 100 sec WXR TEST and WXR test pattern shall be displayed on MFD
  • After several sweeps RADAR OK message shall be displayed in test pattern -> WXR serviceable
  • or Reset WX CP (H1) and WX RDR (H2) circuit breakers on avionics CB panel, then see Action 1

ON MFD 1 ONLY

  • Check, if EGPWS TERRAIN information can be displayed on MFD 1 (one FMS has to be on with GNSS in NAV mode!)
  • If not, check picture bus connection on MFD 1
  • If yes, replace relay 3443-K7 Reset WX CP (H1) and WX RDR (H2) circuit breakers on avionics CB panel

ON MFD 2 ONLY

  • Check, if EGPWS TERRAIN information can be displayed on MFD 2 (one FMS has to be on with GNSS in NAV mode!)
  • If not, check picture bus connection on MFD 2
  • If yes, replace relay 3443-K8 Reset WX CP (H1) and WX RDR (H2) circuit breakers on avionics CB panel
AVIONICS CB PANEL

H1 - WX CP
H2 - WX RDR
