TERR FAIL

TERRAIN FAIL on both MFD1 FIM TASK 34-42-00-810-811 (TR-FIM) TERRAIN FAIL on both MFD2 FIM TASK 34-42-00-810-812 (TR-FIM)

NOTE: MFD1/2 will only show a TERR FAIL message if it does not show a [CHECK ED] message.

NOTE: The enhanced ground proximity warning system (EGPWS) level 1 short or long self−test preamble also generates a “TERR INOP” voice message.

The MFD1/2 TERR FAIL message comes into view for the malfunction conditions that follow:

  • The flight management system (FMS) is not energized, accurate latitude and longitude position data is not supplied.
  • The condition that caused the caution and warning panel (CAWP) GPWS caution light to come on.
  • The attitude and heading reference system (AHRS1 and AHRS2) data is not supplied.
  • The MFD does not receive valid terrain image data through a relay from the enhanced ground proximity warning computer (EGPWC) A453 data bus (terrain data bus) while the terrain MFD is not set off.
  • The range data received from the terrain data bus output is not the same as the calculation made by the MFD while not in the terrain alert mode.




NOTE:
MSG on both MFDs
  • MSG on both MFDs
      • 1. Switch on at least one FMS, check GNSS status on page DATA 2/4 (should be NAV). If one GNSS is NAV then refer to Action 2.
      2. GPWS (A1) circuit breaker on avionics CB pane

  • MSG on MFD 1 only
      • 1. Check, if WXR information can be displayed on MFD 1. If not, check picture bus connection on MFD 1. If yes, replace relay 3443-K7
      2. Reset GPWS (A1) circuit breaker on avionics CB panel

  • MSG on MFD 2 only
      • 1. Check, if WXR information can be displayed on MFD 2. If not, check picture bus connection on MFD 2. If yes, replace relay 3443-K8
      2. Reset GPWS (A1) circuit breaker on avionics CB panel
