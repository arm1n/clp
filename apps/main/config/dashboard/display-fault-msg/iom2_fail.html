IOM2 FAIL

FIM TASK 31-41-00-810-868 (TR-FIM)

The ED shows this message when input/output processor 2 (IOP2) senses that input/output module 2 (IOM2) has malfunctioned.

NOTE: The AVIONICS caution light will also come on.








NOTE:

Reset IOM/IOP 1 and IOM/IOP 2

  • IOM/IOP1 (F7) on avionics CB panel
  • IOM/IOP2 (F6) on avionics CB panel
