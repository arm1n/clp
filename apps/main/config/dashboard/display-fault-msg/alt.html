ALT MESSAGE ON PFD
The fault isolation procedure is for when the primary flight display (PFD) altitude indicator shows the ALT mismatch flag.

The PFD shows the ALT mismatch flag when the flight data processing system (FDPS) senses and air data unit 1 (ADU1) and ADU2 altitude difference that is more than 60 feet at sea level to 180 feet at 27 00 feet.

The PFD flight mode annunciator (FMA) can also show an ALT MISMATCH message.


ALT, ATC MESSAGE ON ARCDU1/2
This fault isolation procedure is for when air traffic control (ATC) area of the audio and radio control display unit 1 (ARCDU1) shows a red ALT message. ARCDU1/2 shows a red ALT message when it does not receive valid ATC mode C data from the set ATC transponder.

The ATC1 or ATC2 (internally by an LED segment on the I/O processor board) can show a related fault that follows:
  • F4H
  • F5H
  • F9H
  • FAH

ALT MESSAGE on Integrated Standby Instrument
This fault isolation procedure is for when the integrated standby instrument (ISI) shows the ALT message.
The ISI shows the ALT message when the ISI senses an internal malfunction of the altitude (ALT) function.


NOTE:
Message may be caused by unintended manual navigation mode change or pitch wheel input on FGCP by pilot.
