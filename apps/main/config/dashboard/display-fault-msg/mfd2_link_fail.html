ED MON FAIL, MFD1

The ED MON FAIL message is usually out of view and comes into view when the set engine display (ED) senses that its very important parameters are not monitored by a different display unit (DU) for more than 30 seconds. The caution and warning panel (CAWP) AVIONICS caution light will also come on.

NOTE: When the MFD1 is set to show ED indications, it is monitored by MFD2.

NOTE: Each display unit (DU) has a feedback output bus that supplies the status of its monitoring function to the set ED as follows:

  • Operating correctly or fail
  • Indication format (PFD1, PFD2, or ED)


MFD2 LINK FAIL, MFD1 (Advisory)

The MFD2 LINK FAIL message is usually out of view and comes into view when conditions are as follows:

  • The set engine display (ED) senses that the MFD2 feedback output bus has malfunctioned for more than 30 seconds and
  • The MFD2 is not set to show PFD or ED indications (very important parameters). The caution and warning panel (CAWP) AVIONICS caution light will not come on.

NOTE: If the MFD2 is set to show PFD indications, the MFD2 LINK FAIL message changes to PFD2 MON FAIL and the CAWP AVIONICS caution light will come on.

NOTE: If the MFD2 is set to show ED indications, the MFD2 LINK FAIL message changes to ED MON FAIL and the CAWP AVIONICS caution light will come on.

NOTE: When the MFD1 is set to show ED indications, it is monitored by MFD2.

NOTE: Each display unit (DU) has a feedback output bus that supplies the status of its monitoring function to the set ED as follows:

  • Operating correctly or fail
  • Indication format (PFD1, PFD2, or ED)
