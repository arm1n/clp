GPS INTEG MESSAGE

FIM TASK 34-50-00-810-803 (TR-FIM)

This fault is set when the sensor of the global positioning system (GPS) indicates that the receiver autonomous integrity monitoring (RAIM) is not available.

NOTE:

  • Switch FMS off and on again.

  • If problem still present after a few minutes check satellite reception on DATA 2/4 page of FMS (should be 4 or more)

  • Possible problem with defective GPS antenna or FMS
