IAS MISMATCH on PFDS

FIM TASK 34-11-00-810-864 (TR-FIM)

The FMA shows an IAS MISMATCH message when the Flight Data Processing System (FDPS) senses an Air Data Unit 1 (ADU 1) and ADU 2 Indicated Airspeed (IAS) difference that is more than 10 KIAS.

NOTE: The FMAs will show only the most important mismatch message and this is the primary mismatch indication.

NOTE: The Air Data Computer (ADC) data must be valid and the Electronic Flight Instrument System (EFIS) ADC SOURCE reversion selector must be set to the NORM position.








NOTE:

1.

Reset ADU CB´s (A6) and (A7) on avionics circuit breaker panel


2.

Reset ADU, AHRS and IOP CB´s on avionics circuit breaker panel:

  • A4 AHR1
  • A9 AHR1
  • A7 ADU1
  • A6 ADU2
  • A3 AHR2
  • A10 AHR2
  • F7 IOP/IOM/FGM1
  • F6 IOP/IOM/FGM2

2.

Check pitot/static system for water ingress, clean drain holes on pitot/static probes, drain pitot/static system
