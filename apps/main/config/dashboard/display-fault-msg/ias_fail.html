IAS FAIL ON PFD

The PFD shows the IAS mismatch flag when the flight data processing system (FDPS) senses and air data unit 1 (ADU1) and ADU2 indicated airspeed (IAS) difference that is more than 10 KIAS). The PFD flight mode annunciator (FMA) will also show an IAS MISMATCH message.

NOTE: The FMAs will show only the most important mismatch message and this is the primary mismatch indication.



IAS FAIL ON MFD1/2

MFD1/2 shows an IAS FAIL message when it is set to show a Primary Flight Display (PFD) reversion and does not receive valid airspeed data from the set Air Data Unit (ADU).

NOTE: The MFD1/2 will also show the IAS FAIL message if the ADU does not receive valid Weight-On-Wheels (WOW) data from the Proximity Sensor Electronics Unit (PSEU) while the airspeed is less than 35 KIAS.



IAS FAIL ON PFD1/2

PFD1/2 shows an IAS FAIL message when it does not receive valid airspeed data from the set Air Data Unit (ADU).

NOTE: The PFD1/2 will also show the IAS FAIL message if the ADU does not receive valid Weight-On-Wheels (WOW) data from the Proximity Sensor Electronics Unit (PSEU) while the airspeed is less than 35 KIAS.



IAS FAIL FLAG ON Integrated Standby Instrument

The ISI shows the IAS message when the ISI senses an internal malfunction of the indicated air speed (IAS) function.
