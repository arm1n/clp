HDG FAIL, MFD1/2 (Warning)

MFD1 shows a HDG FAIL message when it is set to show a primary flight display (PFD) reversion and does not receive valid heading data from the set attitude and heading reference unit (AHRU).

A related AHCP1 or AHCP2 SLAVE warning light also can possibly come on



HDG FAIL, PFD1/2

PFD1 shows a HDG FAIL message when it does not receive valid heading data from the set attitude and heading reference unit (AHRU).

A related AHCP1 or AHCP2 SLAVE warning light also can possibly come on.




NOTE:
  • Re-align AHRU 1 (push ALIGN on AHCP 1)
  • Reset AHRU 1
      • (A4) & (A9) Avionics CB


  • Re-align AHRU 2 (push ALIGN on AHCP 2)
  • Reset AHRU 2
      • (A3) & (A10) Avionics CB
