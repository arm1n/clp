VHF COM FAIL message on ARCDU

FIM TASK 23-11-00-810-804 (TR-FIM)

This fault isolation procedure is for when the very high frequency transceiver 1 (VHF1) display area of the audio and radio control display unit 1 (ARCDU1) shows a FAIL message.

The VHF1 display area of ARCDU1 shows a FAIL message when ARCDU1 does not receive valid data from VHF1.

There are no related central diagnostic system messages.

The VHF1 front panel can show one or more of the fault indications that follow:

  • FAIL IN
  • FAIL LRU

NOTE: A voltage standing wave ratio (VSWR) LED indication is only shown while the VHF transceiver is transmitting and the VSWR is more than 2:1.

The ARCDU will not show a VHF1 FAIL message for this condition.
