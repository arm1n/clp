The PFD1 MON FAIL message is usually out of view and comes into view when conditions are as follows:

  • The set ED senses that the primary flight display 1 (PFD1) feedback output bus has malfunctioned for more than 30 seconds
  • The PFD1 is set to show PFD1 indications
