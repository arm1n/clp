RAD ALT MISMATCH


FIM TASK 34-40-00-810-803 (TR-FIM)

The two FMAs show a RAD ALT MISMATCH message when the flight data processing system (FDPS) senses a radio altimeter 1 (RA1) and RA2 altitude difference that is more than 20 feet. The PFD altitude indicator will show a RA mismatch flag.

NOTE: The audio and radio control display unit (ARCDU) does not show any related central diagnostic system (CDS) messages.
