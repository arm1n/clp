MFD1 MON FAIL

ED MON FAIL, MFD1 FIM TASK 31-61-01-810-808 (TR-FIM)

The ED MON FAIL message is usually out of view and comes into view when the set ED senses that its very important parameters are not monitored by a different Display Unit (DU) for more than 30 seconds. The Caution and Warning Panel (CAWP) AVIONICS caution light will also come on.

NOTE: The ED is monitored by Multi-Function Display 1 (MFD1) and MFD2.

NOTE: Each DU has a feed back output bus that supplies the status of its monitoring function to the set ED as follows:

  • Operating correctly or fail
  • Indication format (PFD1, PFD2, or ED)
