A/F TEST ABORT

A/F TEST ABORT - ENG 1 FIM TASK 61-41-00-810-809 (TR-FIM) A/F TEST ABORT - ENG 2 FIM TASK 61-41-00-810-810 (TR-FIM) O/C of the Propeller
Autofeather and Uptrim System AMM TASK 61-20-00-710-810 (TR-AMM)

NOTE:

During fault isolation for OSG and AF test faults, the tests must be done again after corrective action has been taken. If the tests pass, a fault clear will be necessary to clear the fault in question.

The A/F TEST ABORT powerplant message comes into view when propeller electronic control 1 (PEC1) or PEC2 senses a condition that follows:
  • Loss of the autofeather select signal during the test
  • Condition lever angle (CLA) signal of 850 rpm or greater being present during the test
  • Power lever angle (PLA) lever position at or above flight idle during testing
  • Loss of the autofeather permissive signal from the opposite powerplant
  • Loss of the opposite powerplant torque signal indicating that the opposite powerplant is producing useful power
  • PEC changing lanes during test
The ED POWERPLANT message will not come into view.
The A/F TEST ABORT powerplant message comes into view during an autofeather test.

The autofeather system will do an autofeather test, an uptrim test, and an AUPC test when conditions are as follows:
  • Aircraft is on the ground
  • No. 1 and No. 2 CONDITION lever position is at FUEL OFF or START/FEATHER
  • No. 1 and No. 2 POWER lever position is at DISC
  • The FUEL/ENGINE CONTROL panel AUTOFEATHER SELECT annunciator switch is pushed.

NOTE:
The aircraft on the ground condition is sensed with the proximity sensor electronics unit (PSEU) or the radio altimeter (RA) system.

An A/F TEST IN PROG and then an A/F TEST ABORT, A/F TEST PASSED, or A/F TEST FAIL powerplant message is also shown. If the A/F TEST FAIL powerplant message comes into view, the A/F SELECT and A/F ARM powerplant messages will not come into view.

The ED UPTRIM powerplant message also comes into view and then goes out of view. The engine mode message changes from NTOP to MTOP while the UPTRIM powerplant is in view. Also, the torque bug changes from 90% to 100% while the UPTRIM powerplant message is in view.

NOTE:
The AUTOFEATHER SELECT annunciator switch selection is supplied through the two PECs, PEC1 and PEC2, to their related full authority digital electronic control (FADEC), FADEC1 and FADEC2.

The A/F TEST ABORT powerplant message comes into view when the FADEC supplies valid data to the ED. The A/F TEST ABORT message goes out of view if the data is invalid.

NOTE:
The autofeather test is manually set for take-off only. It will automatically feather the propeller after an engine failure and give power uptrim of the operating engine.
