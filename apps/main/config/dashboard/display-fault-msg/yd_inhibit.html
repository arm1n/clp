YD INHIBIT


FIM TASK 22-11-00-810-843 (TR-FIM)

This fault isolation procedure is for when the flight mode annunciator (FMA) in the primary flight display shows the YD INHIBIT message. The FMA shows this message for a yaw damper (YD) selection when an external condition prevents yaw damper operation.

The PFD shows the YD NOT CENTERED message when the yaw damper actuator unit (YDAU) is not engaged and not in the centre position.

NOTE: If an attitude and heading reference unit (AHRU) does not supply valid or the same attitude data to the two FGMs, the FMA will show a FD ATT DATA INVLD message.

NOTE: If an air data computer (ADC) does not supply valid altitude data to the two FGMs, the FMA will show a FD ADC DATA INVLD message.

NOTE: The YD INHIBIT FMA message will also come into view for the conditions that are not malfunctions as follows:

  • Aircraft attitude is not in the engagement limits
  • YD DISENGAGED message is in view
