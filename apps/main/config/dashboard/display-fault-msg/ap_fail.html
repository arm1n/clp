AP FAIL

FIM TASK 22-11-00-810-808 (TR-FIM)

The FMA shows this message when there is an internal Flight Guidance Module (FGM), Autopilot Pitch Actuator Unit (APAU−P), Autopilot Roll Actuator Unit (APAU−R) or Pitch (P) or Roll (R) inhibit malfunction that prevents Autopilot (AP) operation.

NOTE:
  • If the AP or Yaw Damper (YD) was engaged, an AP DISENGAGED message is shown until cancelled before this message is shown.

  • The FMA in the Primary Flight Display (PFD) can also show the YD NOT CENTERED message.
