DUAL OFF


FIM TASK 22-11-00-810-836 (TR-FIM)

The FMA shows a DUAL OFF message when the display unit (DU) senses a dual flight director (FD) mode malfunction during instrument landing system (ILS) approach that is not category 2. This message is shown when dual FD approach mode is cancelled.

NOTE:
The category 2 ILS mode operates when the decision height (DH) is set less than 200 feet and more than 50 feet.
