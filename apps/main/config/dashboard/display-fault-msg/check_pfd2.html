[CHECK PFD2] message on PFD1 or PFD2 and the MFD2

FIM TASK 31-61-01-810-805 (TR-FIM)

BOLD

The PFDs and MFDs (system or navigation pages) show this message when the MFD1/2 or PFD2/1 senses a difference in PFD1/2 indication for one or more of these parameters:

  • Altitude
  • Airspeed
  • Pitch
  • Roll
  • Heading

NOTE:
If the MFD1 is set to show PFD1/2 messages, or if it fails, it will not monitor PFD1/2 parameters.

NOTE:
If the PFD2 fails, the MFD2 monitors the PFD1. The MFD1 continues to monitor the PFD1.

NOTE:
If the MFD1 is set to show PFD1 messages, it is monitored by the PFD2, or by the MFD2 if the PFD2 fails.
