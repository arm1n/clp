PFD2 LINK FAIL

PFD2 LINK FAIL on ED FIM TASK 31-61-01-810-819 (TR-FIM) PFD2 LINK FAIL on MFD2 FIM TASK 31-61-01-810-857 (TR-FIM)

The PFD2 LINK FAIL message is usually out of view and comes into view when conditions are as follows:

  • The set ED senses that the primary flight display 2 (PFD2) feedback output bus has malfunctioned for more than 30 seconds
  • The PFD2 is not set to show PFD indications (very important parameters)

NOTE: If the PFD2 is set to show PFD indications, the PFD2 LINK FAIL message changes to PFD2 MON FAIL and the CAWP AVIONICS caution light will come on.
