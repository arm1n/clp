FIM TASK 31-61-01-810-842 (TR-FIM)

The green T is shown on ED during the power−on self−test (POST) mode and stays in view for more than one minute, if the POST mode senses a discrepancy. The ED POST mode starts automatically after a long electrical power interruption while the aircraft is on the ground.

An ED WARMING UP message is shown after the POST indication if the ED has sensed a low internal temperature and an internal heater has automatically come on.

NOTE: The EFCP2 PFD OFF, BRT knob is set to the BRT position to energize primary flight display 2 (PFD2) and control its brightness.The engine and system integrated display (ESID) system has a mono mode to view the system page with a composite permanent system data indication when the two multi-function displays (MFDs) malfunction or are set to show PFD, ED or navigation page indications.
