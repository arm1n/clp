ALT FAIL ON Integrated Standby Instrument
This fault isolation procedure is for when the integrated standby instrument (ISI) shows the ALT message.The ISI shows the ALT message when the ISI senses an internal malfunction of the altitude (ALT) function.

ALT FAIL ON PFD
he fault isolation procedure is for when the primary flight display (PFD) altitude indicator shows the ALT mismatch flag. The PFD shows the ALT mismatch flag when the flight data processing system (FDPS) senses and air data unit 1 (ADU1) and ADU2 altitude difference that is more than 60 feet at sea level to 180 feet at 27 00 feet. The PFD flight mode annunciator (FMA) can also show an ALT MISMATCH message.
NOTE: The FMAs will show only the most important mismatch message and this is the primary mismatch indication.

ALT FAIL ON MFD1/2
This fault isolation procedure is for when the multi-function display 1 (MFD1) shows an ALT FAIL message.MFD1/2 shows an ALT FAIL message when it is set to show a primary flight display (PFD) reversion and does not receive valid altitude data from the set air data unit (ADU).

ALT FAIL ON PFD1/2
This fault isolation procedure is for when the primary flight display 1 (PFD1) shows an ALT FAIL message. PFD1 shows an ALT FAIL message when it does not receive valid altitude data from the set air data unit (ADU).

ALT FAIL ON ARCDU1/2
This fault isolation procedure is for when air traffic control (ATC) area of the audio and radio control display unit 1/2 (ARCDU1/2) shows a red ALT message. ARCDU1/2 shows a red ALT message when it does not receive valid ATC mode C data from the set ATC transponder.
NOTE: A related FAIL indication is shown adjacent to the ALT1 or ALT2 indication in the ATC particular page.

The ATC1 or ATC2 (internally by an LED segment on the I/O processor board) can show a related fault that follows:
  • F4H
  • F5H
  • F9H
  • FAH
