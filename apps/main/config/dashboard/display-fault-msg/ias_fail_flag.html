IAS FAIL FLAG on PFD

FIM TASK 34-10-00-810-801 (TR-FIM)

The PFD shows the IAS mismatch flag when the flight data processing system (FDPS) senses and air data unit 1 (ADU1) and ADU2 indicated airspeed (IAS) difference that is more than 10 KIAS).

The PFD flight mode annunciator (FMA) will also show an IAS MISMATCH message.

NOTE: The FMAs will show only the most important mismatch message and this is the primary mismatch indication.
