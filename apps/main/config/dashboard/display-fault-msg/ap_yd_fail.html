AP/YD FAIL

FIM TASK 22-11-00-810-805 (TR-FIM)

The FMA shows the AP/YD FAIL message for one of the conditions that follow:
  • A Flight Guidance Module (FGM1 or FGM2) senses an internal or Yaw Damper Actuator Unit (YDAU) fault that prevents Yaw Damper (YD) operation
  • Input/Output Processor module 1 (IOP1) does not receive a valid status from FGM1 or FGM2.

NOTE:
  • If the Autopilot (AP) or YD was engaged, an AP/YD DISENGAGED message is shown until cancelled before this message is shown.

  • If the YD was engaged, an YD DISENGAGED message is shown until cancelled before this message is shown.

  • The FMA in the PFD can also show the YD NOT CENTERED message.

  • If the message is AP FAIL, do the fault isolation for the AP FAIL (Refer to FIM TASK 22-11-00-810-808 (TR-FIM)).








NOTE:
  • Check AP disengage pushbuttons on control wheels for stuck condition.

  • Reset following circuit breakers on AVIONICS CB PANEL::
    • F7 IOM/IOP/FGM1
    F6 IOM/IOP/FGM2

    E1 APAU PITCH
    E2 APAU ROLL
    E3 YDAU
