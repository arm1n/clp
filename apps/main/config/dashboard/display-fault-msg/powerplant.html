POWERPLANT

FIM TASK 77-31-00-810-835 (TR-FIM)

The set ED shows this message when the Full Authority Digital Engine Control 1 (FADEC1) or FADEC 2 calculates a FADEC or Propeller Electronic Controller (PEC) Time Limited Dispatch (TLD), No Dispatch (ND) condition.

NOTE: The ED is monitored by Multi-Function Display 1 (MFD1) and MFD2.

NOTE: TLD is used to calculate and manage a loss of control system redundancy to meet the Loss of Thrust Control (LOTC) guidelines. The TLD is divided into the categories that follow:

  • Unlimited (UNL)
      • The current powerplant malfunctions do not affect system redundancy, and do not affect dispatch

  • Long Term Dispatch (LTD)
      • The calculation of current malfunctions that affect system redundancy allows dispatch for up to 500 hours from when the discrepancy was sensed.

  • Short Term Dispatch (STD)
      • The calculation of current malfunctions that affect system redundancy allows dispatch for up to 150 hours from when the discrepancy was sensed.

  • No Dispatch (ND)
      • The loss of control system redundancy to meet LOTC guidelines is not acceptable and dispatch is not allowed after the end of the present flight.




Do a FADEC Fault Code Clear of both Engines as follows:

  • 1. Move BOTH POWER LEVER to the DISC detent.

  • 2. Move BOTH CONDITION LEVER to MAX/1020 detent.

  • 3. On the Central Maintenance Panel, set MAINT DISC to ON.

  • 4. Push the RIG TRIM switch for at least 5 seconds

  • 5. Set MAINT DISC to OFF

Move both POWER LEVERS and CONDITION LEVERS to the normal position

FADEC Fault Code Clear AMM TASK 73-21-00-070-801 (TR-AMM)



Do a PEC Fault Code Clear of both Engines as follows:


  • 1. Set BOTH POWER LEVER to the DISC detent.

  • 2. Move BOTH CONDITION LEVER to FUEL OFF position.

  • 3. On the Central Maintenance Panel, set MAINT DISC to ON.

  • 4. Push the RIG TRIM switch for at least 15 seconds

  • 5. Set MAINT DISC switch to OFF

Move both POWER LEVERS and CONDITION LEVERS to the normal position
