OSG TEST ABORT


OSG TEST ABORT on ENG 1 FIM TASK 61-41-00-810-815 (TR-FIM) OSG TEST ABORT on ENG 2 FIM TASK 61-41-00-810-816 (TR-FIM)

The OSG TEST ABORT message will not turn on the POWERPLANT message.

If the last completed OSG test is failed, the POWERPLANT message on the ED and fault code 496/497 must be cleared to pass the OSG test.

A fault clear will not clear this fault code. An OSG test must be satisfactorily completed to clear this fault code. If the fault code is not shown, no more maintenance procedures are necessary.
