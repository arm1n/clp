MISTRIM [TRIM NOSE DN] displayed on PFD


FIM TASK 22-11-00-810-873 (TR-FIM)

The MISTRIM [TRIM NOSE DN] message is usually out of view and comes into view when conditions are as follows:

  • Autopilot (AP) mode is set and the flight guidance module (FGM) senses that the autopilot pitch actuator unit (APAU-P) torque is more than the threshold (mistrim)
  • Torque sensing in the APAU-P has malfunctioned.

NOTE: The APAU-P uses current limiting to limit the maximum torque that can be supplied to the flight controls.

NOTE: The MISTRIM [NOSE DN] message is not shown when the AP mode is not set, or when the AP mode is set and the tactile control steering (TCS) mode is active.

NOTE: The MISTRIM [NOSE DN] message shows that the control column is mistrimmed, and that there can be much more than usual control column force and pitch change when the AP mode is disengaged.
