PITCH message on PFD


FIM TASK 34-20-00-810-805 (TR-FIM)

The EADI shows the PITCH mismatch flag when the flight data processing system (FDPS) senses an attitude and heading reference unit 1 (AHRU1) and AHRU2 pitch difference that is more than three degrees.

The flight mode annunciator (FMA) in the primary flight display (PFD) can show a PITCH MISMATCH message

NOTE: The FMAs will show only the most important mismatch message and this is the primary mismatch indication.
