WOW/IOP1 FAIL (ground only)


WOW/IOP1 FAIL (Advisory) FIM TASK 31-41-00-810-860 (TR-FIM)

This fault isolation procedure is for when the engine display (ED) shows the WOW/IOP1 FAIL message. The ED shows this message when the input/output processor 1 (IOP1) senses different proximity sensor electronics unit (PSEU) weight on wheels 1 (WOW1) and WOW2 inputs. The caution and warning panel (CAWP) AVIONICS caution light will also come on.

NOTE:
This indication is shown on the ground only because the IOP1 cannot do a power-on self-test (POST).

NOTE:
The CAWP WT ON WHEELS caution light will also come on.
