IOP BAD CONF (Status)

FIM TASK 31-41-00-810-863 (TR-FIM)

The set ED shows this message when an input/output processor module (IOP1 or IOP2) senses that it does not receive valid aircraft configuration data from the two aircraft configuration modules (ACM1 and ACM2), or that the data is different.

NOTE: The IOP BAD CONF message can come into view only after a IOP power - on self - test (POST).

NOTE: The caution and warning panel (CAWP) AVIONICS caution light will also come on
