ROLL on PFD

FIM TASK 34-20-00-810-807 (TR-FIM)

The EADI shows the ROLL mismatch flag when the flight data processing system (FDPS) senses an attitude and heading reference unit 1 (AHRU1) and AHRU2 roll difference that is more than three degrees.

The flight mode annunciator (FMA) in the primary flight display (PFD) can show a ROLL MISMATCH message.

NOTE: The FMAs will show only the most important mismatch message and this is the primary mismatch indication.








NOTE:

Reset AHRU 1 and AHRU 2 on Avionics CB Panel

  • (A3) AHRU 2
