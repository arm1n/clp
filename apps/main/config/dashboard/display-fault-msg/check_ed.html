CHECK ED

FIM TASK 31-61-01-810-807 (TR-FIM)

This fault isolation procedure is for when the engine display (ED) or multifunction displays (MFDs) show the [CHECK ED] message.

The ED or MFDs (system or navigation pages) show this message when an MFD senses a primary ED indication discrepancy for one or more of the parameters that follow:

  • Torque
  • High pressure rotor speed (NH)
  • NH red limit
  • Propeller speed (NP)
  • Internal turbine temperature (ITT)
NOTE: If MFD 1 or MFD 2 is set to show ED indications, it will be monitored by the other MFD.

    Reset Display Unit circuit breakers MFD 1 (B7), MFD 2 (B6), ESID (B5) and ESID (B11) on avionics CB panel
