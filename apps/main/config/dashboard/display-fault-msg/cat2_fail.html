CAT 2 FAIL MESSAGE ON MFD1/2

This fault isolation procedure is for when multifunction display 1/2 (MFD1/2) shows the CAT 2 FAIL message.

MFD1/2 shows a CAT 2 FAIL message when it is set to show a primary flight display (PFD) reversion and the display unit (DU) senses a dual flight director (FD) mode malfunction during a category 2 instrument landing system (ILS) approach. This message is shown when dual FD approach mode is cancelled while the decision height (DH) is set less than 200 feet and more than 50 feet.

CAT 2 FAIL MESSAGE ON PFD1/2

This fault isolation procedure is for when the flight mode annunciator (FMA) in primary flight display 1/2 (PFD1/2) shows the CAT 2 FAIL message.

The FMA shows a CAT 2 FAIL message when the display unit (DU) senses a dual flight director (FD) mode malfunction during a category 2 instrument landing system (ILS) approach. This message is shown when dual FD approach mode is cancelled while the decision height (DH) is set less than 200 feet and more than 50 feet.
