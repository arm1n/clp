AP INHIBIT


FIM TASK 22-11-00-810-842 (TR-FIM)
The FMA shows this message for an Autopilot (AP) selection when an external condition prevents autopilot operation.

NOTE:
  • If an Attitude and Heading Reference Unit (AHRU) does not supply valid or the same attitude data to the two Flight Guidance Module (FGM), the FMA will show a FD ATT DATA INVLD message.
  • If an Air Data Computer (ADC) does not supply valid or the same altitude data to the two FGMs, the FMA will show a FD ADC DATA INVLD message.
  • The PFD 1 and PFD 2 also show this message when the FGM 1 or FGM 2 senses different Weight on Wheels 1 (WOW1) and WOW2 inputs from the Proximity Sensor Electronics Unit (PSEU).
  • The WOW/IOP1 FAIL or WOW/IOP2 FAIL message on the Engine Display (ED) can also come in view.

The AP INHIBIT FMA message will also come into view for the conditions that are not malfunctions as follows:
  • Aircraft is not airborne
  • Aircraft attitude is not in the engagement limits
  • Manual pitch trim selection
  • AP DISENGAGE message is in view
  • Tactile Control Steering (TCS) mode is active
  • Stick shaker or pusher is in operation

  • Check elevator trim, TCS and AP disengage switches on control columns for stuck condition

  • Check, if all air data information is valid (altitude, IAS and SAT)
