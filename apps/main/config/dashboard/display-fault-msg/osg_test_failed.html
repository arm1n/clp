OSG TEST FAILED

OSG TEST FAILED on ENG 1 FIM TASK 61-41-00-810-819 (TR-FIM) OSG TEST FAILED on ENG 2 FIM TASK 61-41-00-810-820 (TR-FIM)

The OSG TEST FAILED message will have ND fault code latched in the Propeller Electronic Controller 1/2 (PEC1/PEC2). If the OSG test is not completed, a OSG TEST ABORT message will come.

The OSG TEST FAILED message comes into view when a fault condition is sensed during an overspeed governor test. The ED POWERPLANT message will come into view.

The OSG TEST FAILED message comes into view when the PEC1 senses a condition that follows during an overspeed governor test :

  • Blade angle is more than flight fine stop tolerance (between 13 and 19 degrees).
  • NP is more than 940 RPM.
