AUTO TRIM FAIL
The FMA shows this message when a flight guidance module (FGM) does not receive valid data from the two flap position indication unit (FPIU) channels.

NOTE: If a pitch trim motion invalid occurs during flight, the AUTO TRIM FAIL message is shown on ground when the AP and flap pitch trim is disengaged.
