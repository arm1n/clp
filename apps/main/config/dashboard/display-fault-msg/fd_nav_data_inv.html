FD NAV DATA INVLD on PFDs

FIM TASK 22-11-00-810-840 (TR-FIM)

The two FMAs show this message for a navigation (NAV), approach (APPR), or back course (BC) flight director (FD) selection if the set NAV source does not supply valid very high frequency omnirange (VOR) bearing, localizer (LOC) deviation, glidesope (GS) deviation, or frequency data to the related flight guidance module (FGM).
