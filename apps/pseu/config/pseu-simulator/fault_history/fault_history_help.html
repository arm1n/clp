Fault History Menu Item

The “Rigging Help” sub-menu provides assistance during the installation and maintenance of individual Proximity Sensors installed on the aircraft.

PROXIMITY SENSOR ABBREVIATION
  • NGDRCL (NOSE GEAR DOORS CLOSED)
FLIGHT LEG
  • The number of the flight leg in which the fault occurred

    • Leg #0 is the current flight leg....up to #31.

TARGET CONDITION
  • UNREASONABLE FAR (Valid Target Near condition)

    • If one proximity sensor does not match the aircraft mode, it is flagged as an “Unreasonableness” fault. For example, if the aircraft is in normal cruise and an uplock sensor becomes target “far”, a fault is logged. This is because a target far is not a “reasonable” condition for this sensor.
