Fault History Menu Item

The “Fault History” sub−menu shows faults that occurred during previous flight legs and since the last Fault History Erase.

The data that follows is also included:

  • The number of the flight leg in which the fault occurred (leg #00 is the current flight leg....up to #31)
  • An indication to show if the fault is intermittent.

The software can show the full list of reported faults. You will see only Flight Legs which contain faults.
