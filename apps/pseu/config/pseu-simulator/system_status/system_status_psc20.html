System Status Menu Item

Use this Menu to check the inductance values of the proximity sensors in acc. with AMM TASK 52-51-11-000-801 - (TR-AMM).
There are 20 landing-gear proximity sensors and 11 fuselage door proximity sensors.

PROXIMITY SENSOR ABBREVIATION
  • LGWOW2 (Left Gear Weight On Wheels #2)
TARGET CONDITION
  • TARGET NEAR
  • TARGET FAR
  • TOO CLOSE or TOO FAR
INDUCTANCE VALUE
  • 08.367 (Inductance values of this Proximity Sensor)

    • Nominal Near 8.617 (9.037 to 8.456 Rigging Gap Tolerance) THESE VALUES ARE NOT UNDER REVISION
NOTE: An inductance value for a correctly adjusted proximity sensor that is nearer to the Nominal Near value (that is not in the parenthesis) is preferred. NOTE: A rigging fault is also recorded in the PSEU by the RIGGING HISTORY function. The PSEU records the rigging fault by the flight leg in which the fault occurred.
