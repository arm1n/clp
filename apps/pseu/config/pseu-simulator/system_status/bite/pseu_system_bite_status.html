System BITE Status Menu Item

The System Status menu and sub−menus give the full status of all discrete inputs, discrete outputs and proximity sensors for each PSC card and for the BITE card. You can use this for detailed aircraft level troubleshooting.

Find the following information on this Page:

  • Bite Software Version (BITE_SOFTWARE and VERSION_REV)
  • PSEU Operating Mode − Weight−On−Wheels (WEIGHT_ON_WHEELS)
  • PSEU Operating Mode − Gear Down and Locked (GEAR_DOWN_AND _LOCKED)
  • PSEU Operating Mode − Gear In Transit (GEAR_IN_TRANSIT)
  • PSEU Operating Mode − Gear Up and Locked (GEAR_UP_AND LOCKED)
  • PSEU Power Input − Left Main 1 (ACTIVE or FAILED)
  • PSEU Power Input − Left Main 2 (ACTIVE or FAILED)
  • PSEU Power Input − Right Main (ACTIVE or FAILED)
  • PSEU Power Input − Left Essential 1 (ACTIVE or FAILED)
  • PSEU Power Input − Left Essential 2 (ACTIVE or FAILED)
  • PSEU Power Input − Right Essential (ACTIVE or FAILED)
