Removal of the Engine from the Nacelle (General)


FOR REFERENCE USE ONLY!

Use these instructions only as a reference

Install the tail stand (GSB0700024) to the aircraft.

If you remove one or both Engines add and distribute ballast in acc with AMM TASK 72-00-00-890-803 (TR-AMM)
NOTE:
Download the data from the EMU to a laptop computer. This will allow the operator to retain and review the EMU historical data as necessary.


De-energize the electrical system Refer to AMM TASK 24-00-00-861-802 (TR-AMM) Obey all the electrical/electronic safety precautions Refer to AMM TASK 24-00-00-910-801 (TR-AMM) Obey all the electrostatic discharge safety precautions Refer to AMM TASK 24-00-00-910-802 (TR-AMM)
