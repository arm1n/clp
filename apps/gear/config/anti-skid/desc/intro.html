ANTI−SKID SYSTEM - General Description

The anti-skid system controls hydraulic pressure to the four main wheel brake units to supply individual wheel anti−skid protection. The brake pedals control hydraulic pressure from the No. 1 hydraulic system through the brake control valve to the anti−skid control valves.

The anti−skid control valves control hydraulic pressure to the fuse/shuttle valves and the brake units. When the anti−skid system senses a skid condition, it sends a signal to the applicable skid control valve.

The skid control valve releases the brake pressure through the anti−skid return line. The anti-skid system supplies individual wheel anti−skid control for each main wheel. The anti−skid system operates with the normal brake system.
